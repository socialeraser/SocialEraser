Hilfe & FAQ

Zuletzt aktualisiert: 20. Juni 2026

Schnellstart

  1. Installiere SocialEraser aus dem Download-Bereich auf unserer Startseite.
  2. Klicke auf das SocialEraser-Symbol in der Chrome-Symbolleiste, um das Side Panel zu öffnen.
  3. Öffne die Plattform, die du bereinigen möchtest (X, TikTok usw.) und melde dich bei deinem Konto an.
  4. Wähle im Side Panel aus, welche Elemente du für deine Plattform bereinigen möchtest.
  5. Klicke auf Bereinigung starten. Fortschrittsbalken und Logs erscheinen live.
  6. Nutze jederzeit Pause / Fortsetzen / Stopp.

FAQ

Ist SocialEraser sicher? Wird mein Konto gesperrt?

SocialEraser führt dieselben Aktionen aus, die du auch manuell ausführen würdest (Klicken auf „Löschen", „Unlike", „Lesezeichen entfernen", „Entfolgen"). Da die Aktionen echte Browser-Events sind, sehen sie für die Plattform genauso aus wie manuelle Klicks. Allerdings können die automatisierten Systeme von X / Twitter dennoch Aktivität mit hohem Volumen erkennen. Wir empfehlen:

  • Halte ein gleichmäßiges Tempo — SocialEraser pausiert automatisch bei erkanntem Rate-Limit.
  • Vermeide es, die Erweiterung viele Stunden am Stück laufen zu lassen.
  • Verwende die Datums- / Stichwortfilter, um in kleineren, zielgerichteten Chargen zu bereinigen.

Du übernimmst die volle Verantwortung für die Einhaltung der Nutzungsbedingungen von X. Details findest du in unseren Nutzungsbedingungen.

Warum werden einige meiner Tweets / Likes nicht gelöscht?

Häufige Gründe:

  • Angeheftete Tweets — Angeheftete Tweets können nicht über die Benutzeroberfläche gelöscht werden. Hefte sie zuerst ab.
  • Retweets von Konten, die gesperrt wurden — diese können nicht über die Benutzeroberfläche rückgängig gemacht werden.
  • Tweets von einem anderen Konto — stelle sicher, dass du im richtigen Konto angemeldet bist.
  • UI hat sich geändert — X.com aktualisiert seine Benutzeroberfläche häufig. Klicke im Side Panel auf das Aktualisieren-Symbol, um die neueste Selektor-Konfiguration zu laden.

Kann ich Direktnachrichten (DMs) löschen?

Nein. Das Löschen von DMs bei X / Twitter erfordert einen echten nativen Rechtsklick (Kontextmenü), was durch die event.isTrusted-Prüfung des Browsers geschützt ist. Per JavaScript eingefügte Events werden abgewiesen. Der einzige Weg, dies zu umgehen, würde die Berechtigung chrome.debugger erfordern, die wir nicht anfordern werden, da sie eine starke Vertrauenswarnung beim Nutzer auslöst. Um DMs zu löschen, verwende die native Oberfläche von X manuell.

Wo werden meine Daten gespeichert?

100% auf deinem Gerät, im lokalen Speicher von Chrome. Die Erweiterung hat keinen Server. Eine Deinstallation der Erweiterung löscht dauerhaft alle gespeicherten Daten. Vollständige Details findest du in unserer Datenschutzerklärung.

Welche Sprachen werden unterstützt?

Aktuell: English, 简体中文, 日本語, 한국어, Português, Español, Deutsch, Français. Wechsle über das Flaggen-Symbol im Side Panel.

Funktioniert SocialEraser auf dem Smartphone?

Eine Android-Version (Capacitor) ist in Entwicklung. iOS ist für die Zukunft geplant. Trage dich auf der Warteliste ein unter support@socialeraser.app.

Was ist mit TikTok / YouTube / Facebook?

Kommt bald. Das Projekt befindet sich in aktiver Entwicklung. Folge unserem GitHub für Updates.

Wie deinstalliere ich die Erweiterung?

Rechtsklick auf das SocialEraser-Symbol in der Chrome-Symbolleiste → „Aus Chrome entfernen". Dadurch werden alle lokal gespeicherten Daten gelöscht.
