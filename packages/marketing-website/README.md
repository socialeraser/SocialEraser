# packages/marketing-website

Marketing website for [socialeraser.app](https://socialeraser.app).

Static HTML/CSS (no build step yet). Deployed to Cloudflare Pages via
`scripts/deploy-website.sh`.

Files:
- `index.html` — landing page
- `privacy.html`, `terms.html` — legal
- `help.html` — FAQ / how-to
- `assets/styles.css` — site styles
- `CNAME`, `_headers` — Cloudflare Pages config
