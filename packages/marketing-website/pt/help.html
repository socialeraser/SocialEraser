Ajuda & FAQ

Última atualização: 20 de junho de 2026

Início rápido

  1. Instale o SocialEraser pela seção de download da nossa página inicial.
  2. Clique no ícone do SocialEraser na barra de ferramentas do Chrome para abrir o Painel Lateral.
  3. Abra a plataforma que você quer limpar (X, TikTok, etc.) e faça login na sua conta.
  4. No Painel Lateral, selecione quais itens limpar para sua plataforma.
  5. Clique em Iniciar limpeza. A barra de progresso e os logs aparecem ao vivo.
  6. Use Pausar / Retomar / Parar a qualquer momento.

FAQ

O SocialEraser é seguro? Minha conta pode ser banida?

O SocialEraser executa as mesmas ações que você faria manualmente (clicar em "Excluir", "Desfazer curtida", "Remover marcador", "Deixar de seguir"). Como as ações são eventos reais do navegador, elas parecem idênticas a cliques manuais. Porém, os sistemas automatizados do X / Twitter ainda podem detectar atividade de alto volume. Recomendamos:

  • Manter um ritmo constante — o SocialEraser pausa automaticamente ao detectar limites de taxa.
  • Evitar rodar a extensão por muitas horas seguidas.
  • Usar os filtros de data / palavra-chave para limpar em lotes menores e direcionados.

Você assume toda a responsabilidade pelo cumprimento dos Termos de Serviço do X. Veja nossos Termos de Serviço para detalhes.

Por que alguns dos meus tweets / curtidas não estão sendo apagados?

Razões comuns:

  • Tweets fixados — tweets fixados não podem ser apagados pela interface. Desafixe-os primeiro.
  • Retweets de contas que foram suspensas — não é possível desfazê-los pela interface.
  • Tweets de uma conta diferente — verifique se você está logado na conta correta.
  • UI mudou — o X.com atualiza a interface com frequência. Clique no ícone de atualizar no Painel Lateral para buscar a configuração de seletores mais recente.

Posso apagar Mensagens Diretas (DMs)?

Não. A exclusão de DMs do X / Twitter exige um clique direito nativo real (menu de contexto), que é protegido pela verificação event.isTrusted do navegador. Eventos injetados por JavaScript são rejeitados. A única forma de contornar isso exigiria a permissão chrome.debugger, que nós não solicitamos porque ela dispara um forte aviso de confiança ao usuário. Para apagar DMs, use a interface nativa do X manualmente.

Onde meus dados ficam armazenados?

100% no seu dispositivo, no armazenamento local do Chrome. A extensão não tem servidor. Desinstalar a extensão apaga permanentemente todos os dados armazenados. Veja nossa Política de Privacidade para detalhes completos.

Quais idiomas são suportados?

Atualmente: English, 简体中文, 日本語, 한국어, Português, Español, Deutsch, Français. Alterne pelo ícone de bandeira no Painel Lateral.

O SocialEraser funciona no celular?

Uma versão para Android (Capacitor) está em desenvolvimento. iOS está planejado para o futuro. Inscreva-se na lista de espera em support@socialeraser.app.

E quanto a TikTok / YouTube / Facebook?

Em breve. O projeto está em desenvolvimento ativo. Acompanhe nosso GitHub para atualizações.

Como faço para desinstalar?

Clique com o botão direito no ícone do SocialEraser na barra de ferramentas do Chrome → "Remover do Chrome". Isso apaga todos os dados armazenados localmente.
