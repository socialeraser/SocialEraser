ヘルプ & FAQ

最終更新日: 2026 年 6 月 20 日

クイックスタート

  1. ホームページのダウンロード セクション から SocialEraser をインストールします。
  2. Chrome のツールバーにある SocialEraser アイコンをクリックして、サイド パネルを開きます。
  3. クリーンアップしたいプラットフォーム (X、TikTok など) を開き、アカウントにログインします。
  4. サイド パネルで、プラットフォームごとにクリーンアップする項目を選択します。
  5. Start Cleanup をクリックします。プログレス バーとログがリアルタイムで表示されます。
  6. いつでも Pause / Resume / Stop を使用できます。

FAQ

SocialEraser は安全ですか? アカウントが BAN されませんか?

SocialEraser は手動で行う操作 (「削除」「いいね解除」「ブックマーク解除」「フォロー解除」のクリック) と同じことを行います。実際のブラウザ イベントとして実行されるため、手動のクリックと区別がつきません。ただし、X / Twitter の自動システムは依然として大量操作を検出する可能性があります。以下のことを推奨します。

  • 一定したペースで実行する — SocialEraser はレート制限を検出すると自動で一時停止します。
  • 拡張機能を長時間連続で実行しないでください。
  • 日付 / キーワード フィルターを使って、小さく対象を絞ったバッチでクリーンアップしてください。

X の利用規約遵守について、すべての責任はユーザーが負うものとします。詳細は 利用規約 をご覧ください。

一部のツイート / いいねが削除されないのはなぜですか?

よくある理由:

  • ピン留めされたツイート — ピン留めツイートは UI から削除できません。先にピン留めを外してください。
  • 停止されたアカウントからのリツイート — UI からは取り消せません。
  • 別のアカウントからのツイート — 正しいアカウントにログインしていることを確認してください。
  • UI の変更 — X.com は UI を頻繁に更新します。サイド パネルの再読み込みアイコンをクリックして、最新のセレクター設定を取得してください。

ダイレクト メッセージ (DM) を削除できますか?

できません。X / Twitter の DM 削除にはブラウザの event.isTrusted チェックで保護された本物のネイティブ右クリック (コンテキスト メニュー) が必要です。JavaScript で注入されたイベントは拒否されます。これを回避するには chrome.debugger 権限が必要ですが、これは強力なユーザー信頼警告をトリガーするため、私たちは 決して 要求しません。DM を削除するには、X のネイティブ インターフェースを手動で使用してください。

データはどこに保存されますか?

100% あなたの端末の Chrome ローカル ストレージに保存されます。拡張機能にはサーバーがありません。拡張機能をアンインストールすると、保存されたすべてのデータが完全に削除されます。詳細は プライバシー ポリシー をご覧ください。

対応言語は?

現在: English, 简体中文, 日本語, 한국어, Português, Español, Deutsch, Français。サイド パネルのフラグ アイコンから切り替えできます。

SocialEraser はモバイルで動作しますか?

Android (Capacitor) 版を開発中です。iOS も将来的に計画されています。ウェイトリストへの登録は support@socialeraser.app までご連絡ください。

TikTok / YouTube / Facebook は?

近日対応予定です。プロジェクトは活発に開発されています。 GitHub をフォローして最新情報を確認してください。

アンインストール方法は?

Chrome ツールバーの SocialEraser アイコンを右クリック → 「Chrome から削除」。これにより、ローカルに保存されたすべてのデータが削除されます。

バグの報告

バグを見つけた場合や機能リクエストがある場合は、 GitHub で Issue を開くか、support@socialeraser.app までメールしてください。
