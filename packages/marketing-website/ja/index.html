基本

SocialEraser は具体的に何をしますか？

選択したソーシャル メディア プラットフォーム上のコンテンツ（投稿、いいね、ブックマーク、フォロー、リポストなど）を一括削除します。何を削除するか、フィルターを設定すれば、ブラウザ内で項目を 1 つずつ処理し、一時停止 / 再開のステート マシンを備えています。

本当に無料ですか？

はい。公開中無料、アカウント不要。投げ銭は歓迎しますが、必須ではありません。ペイウォールや回数制限、アップグレードの催促は一切ありません。

アカウントを作成する必要がありますか？

いいえ。アカウント、サインアップ、メールは一切不要。インストールして使うだけ。

対応ブラウザは？

現時点では Chrome と Edge。Firefox と Safari は 2026 年のロードマップにあります。

クリーンアップ中はブラウザを開いたままにしておく必要がありますか？

クリーンアップが実行されているブラウザ タブは開いたままにしておく必要がありますが、最小化することはできます。Chrome を閉じると、実行は一時停止し、再開時に停止した場所からそのまま続行されます。サイド パネルはタブから切り離して表示を維持できます。

プライバシーとセキュリティ

パスワードを要求されますか？

一切要求しません。SocialEraser はソーシャル メディアのパスワードを一切聞きません。すでにログイン済みのブラウザ セッション内で動作するため、閲覧する手段がなく、あなたのアカウントを見ることはできません。

私のデータは収集されますか？

いいえ。データを受け取るサーバーはありません。拡張機能は完全にクライアント サイドで動作します。chrome.storage に保存されるのは設定（言語、日次カウンター）のみ — ワンクリックで削除できます。

インストールしても安全ですか？

拡張機能は Chrome ウェブ ストアに掲載される前に Google による審査を受けています。ソース コードは GitHub でオープン ソース（MIT ライセンス）として公開されているため、誰でも監査できます。すべてのリモート スクリプトをブロックする厳格なコンテンツ セキュリティ ポリシーを使用しています。

使用中にアカウントが BAN される可能性は？

SocialEraser は人間に近い速度で通常のクリックをシミュレートします。プラットフォームにはレート制限があり、私たちはそれを尊重します。とはいえ、どのプラットフォームとも提携していないため、一括ツールの使用はご自身の判断で行ってください。他のサードパーティ ツールにも同じことが言えます。

拡張機能がサーバーと通信していないことを確認できますか？

はい。拡張機能の実行中に Chrome DevTools → Network タブを開けば、サードパーティ ドメインへの送信リクエストがゼロであることがわかります。ネットワーク アクティビティはあなたのブラウザとクリーンアップ対象のソーシャル プラットフォーム間のみ — あなたが手動で行うのと同じリクエストです。完全なソースは GitHub で誰でも監査できます。

仕組み

一括削除は実際にどう動作しますか？

拡張機能はプラットフォームのネイティブ UI（例：x.com）を開き、各項目のアクション メニューを見つけて、削除をクリックして確認します。あなたが手動で行うのと同じ — ただしより高速で疲れません。

どんなフィルターが使えますか？

日付範囲（X より前 / 後の投稿を削除）、キーワード（「Y」を含む投稿のみ削除）、メディア タイプ（画像のみ、テキストのみ、動画のみ）、アカウント（このハンドルからの投稿のみ）。フィルターの組み合わせは OR ではなく AND として機能します。

処理を一時停止・再開できますか？

はい。ブラウザを閉じた後でも、いつでも一時停止 / 再開できます。状態は chrome.storage.local に保存されるため、再起動後も保持されます。翌日に戻って、停止した場所からそのまま続行できます。

どのくらい速いですか？

1 件あたり平均 1 秒未満。1000 件の処理は約 15 分です。正確な見積もりには上部の 時間計算ツール をご利用ください。

削除前にエクスポートできますか？

はい。X ティアの拡張機能には、削除前にすべての項目の archive.json を保存する「Export archive」オプションがあります。他のプラットフォームにも順次追加中です。

クリーンアップ中にインターネットが切れた場合はどうなりますか？

拡張機能は切断を検知して実行を一時停止し、「再接続中…」インジケーターを表示します。ネットワークが復旧したら、最後に完了した項目から処理を再開します。データは一切送信・保存されず、再開状態はお使いのブラウザのローカル ストレージのみに保持されます。

クリーンアップを取り消すことはできますか？

いいえ。プラットフォーム上で投稿が削除されると、それで終わりです。クリーンアップを実行する前に「Export archive」オプション（利用可能な場合）を使用して JSON バックアップを保存し、削除した項目の記録を残しておくことをおすすめします。

代替ツールとの比較

SocialEraser vs TweetDelete？

TweetDelete は X API を使用し、OAuth 認証が必要です。サーバー サイドで動作するため、データはあなたの端末から出ます。SocialEraser はブラウザ内で動作し、認証情報を要求せず、X だけでなく 5 つのプラットフォームに対応します。

SocialEraser vs Semiphemeral？

Semiphemeral は経過時間ベースの自動削除で、X か月以上古い投稿をスケジュールで削除します。SocialEraser はオンデマンドの一括削除で、何を、いつ、どの程度清理するかをユーザーが選択します。別の目的のための別のツールです。

SocialEraser vs Redact？

Redact は API 経由で投稿を一括削除する有料の開発者向けツールです。API キーと開発者アカウントが必要です。SocialEraser は無料で、API アクセス不要で、X 以外のプラットフォームにも対応します。

SocialEraser vs 手作業？

1000 件を手動で削除すると、約 3 時間の連続クリックが必要です。SocialEraser は同じ作業を約 15 分で完了し、疲れることも、確認ダイアログを見落とすことも、深夜 2 時に誤ったボタンをクリックすることもありません。同じ動作を、自動で実行するだけです。

プラットフォーム別

TikTok / YouTube / Instagram / Facebook はなぜ未対応？

各プラットフォームには固有の UI パターン、レート制限、ボット対策があります。私たちは 1 つずつプラットフォームを出荷し、信頼性が確保されるまで調整します。X が最初に出荷されたのは、最も明確な需要があったためです。他はキューに入っています — ウェイトリスト に参加して通知を受け取ってください。

モバイル（iOS / Android）アプリも無料ですか？

はい。Android と iOS のアプリも公開中無料です。回数制限やアカウント登録は不要です。

複数のアカウントで SocialEraser を使えますか？

Chrome では、アカウントごとに別のブラウザ プロファイルを使用できます。1 つのプロファイル内での複数アカウント対応はロードマップにあります。

最初にプラットフォームにログインしておく必要がありますか？

はい。SocialEraser はすでにログイン済みのブラウザ セッション内で動作します。ログイン操作、クッキー保存、認証トークンへの干渉は一切行いません。セッションの有効期限が切れた場合は、ログインし直して実行を再開してください。
