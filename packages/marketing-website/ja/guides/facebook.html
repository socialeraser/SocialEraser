Facebookの投稿を一括削除する方法(ステップバイステップガイド)

なぜFacebookの投稿を一括削除するのか?

ほとんどの人は、いつかクリーンなFacebookが欲しくなる瞬間があります — リブランド、転職、新しい街への引っ越し、心機一転、あるいは単にタイムラインが2012年の自分の博物館であることに気付く瞬間です。問題は、Facebookの公式の「投稿を削除」ワークフローが1件ずつしか行えないことです。一括削除も、バッチ操作も、キーワードフィルターも、期間指定のクリーンアップも組み込まれていません。

200件なら午後のクリックで済みます。2,000件なら1週間。3,000件なら、何かしらの助けなしには事実上不可能です。

そこで活躍するのがFacebook用SocialEraserです。ブラウザで動作し、人間の速度で通常のクリックをシミュレートし、日付、キーワード、投稿タイプ、友達非活動でフィルターできます。ルールを設定し、スタートをクリックし、あとは離れるだけ。このガイドではその全工程を解説します。

始める前に: データをバックアップする

このステップは最も重要なので、最初の実行前に必ず行ってください。Facebookの投稿を一度完全に削除したら復元できません — ごみ箱も、元に戻すボタンもありません。Facebookの「アーカイブ」機能は復元可能な状態を提供しますが、アプリ内のアーカイブボタンを使用する必要があります。

投稿を取り戻す(削除後でも)公式の方法は、始める前にFacebookから「情報をダウンロード」アーカイブをリクエストすることです。アーカイブにはすべての投稿、写真、コメント、いいね、友達、メッセージが個別のファイルとして含まれます。

  1. ブラウザで facebook.com を開き、ログインします。
  2. 直接 facebook.com/dyi にアクセスします。
  3. JSON + HTML 形式を選び、利用可能なすべての情報を選択してから、ダウンロードをリクエストをクリックします。
  4. Facebookからダウンロードリンクがメールで届くのを待ちます。通常は数時間、遅くとも数日かかります。

ヒント: すべてを削除する予定でも、最初にこのアーカイブをリクエストしてください。無料で、あなた自身のデータであり、もし気が変わったときに戻る手段を残してくれます。ボーナスとして、最初の実行ではアプリ内のアーカイブオプション(完全削除ではなく)を使用することもできます — アーカイブされた投稿はアクティビティログからいつでも復元できます。アーカイブがより安全なデフォルトです。

ステップバイステップ: すべてのFacebookの投稿を一括削除する

全体の流れはセットアップに約60秒、実際の削除はバックグラウンドで実行されます。

1

Facebook用SocialEraserをインストールする

Facebook用SocialEraserのページを開き、Facebookウェイトリストに登録をクリックします。拡張機能は現在発売前です — ウェイトリストに登録すると、配信時にいち早くアクセスできます。(本日利用可能な X用SocialEraser をインストールすることもできます。これはすでに完全にリリースされています。)

2

facebook.comを開いてログインする

SocialEraserをインストールしたブラウザで facebook.com にアクセスし、通常通りアカウントにログインしてください。拡張機能はパスワードを扱いません — Facebook自身のログイン機構が使われます。SocialEraserは、認証後のページ上でのみ動作します。

ツールバーにあるSocialEraserアイコンをクリックしてサイドパネルを開きます。ログインが確定すると、上部のステータスインジケーターが黄色から緑色に変わります。

3

クリーンアップ対象を選ぶ

サイドパネルで、削除したい対象をチェックします。1つだけ選んでも、組み合わせても構いません:

  • タイムライン投稿 — 自分のステータス、写真投稿、リンクシェア、チェックイン
  • 写真 — アップロードした写真と写真アルバム
  • いいね — いいねしたすべての投稿とページ
  • 保存アイテム — 非公開の保存投稿とリンク
  • 友達 — フレンド追加したアカウント
  • コメント — 他の人の投稿に対する自分のコメント
  • メッセージ — 自分が開始したMessengerスレッド
  • 管理ページ — 自分が管理するページの投稿

1種類(例えばタイムライン投稿だけ)だけをクリーンアップしたい場合は、そのチェックボックスだけをオンにします。拡張機能は他の対象を完全にスキップします。

4

フィルターを設定する(任意だが推奨)

フィルターこそが、精密な一括削除と、すべてを更地にする荒っぽい削除を分けるポイントです。4つのフィルターについては次のセクションで詳しく解説しますが、最も一般的な設定は以下のとおりです:

  • 開始日: 最近の投稿は残し、特定の日付以前のものをすべて削除
  • キーワード: キャプションに特定の単語を含む投稿だけを削除
  • 投稿タイプ: ステータスのみ、写真のみ、リンクのみ、またはチェックインのみ
  • 友達非活動: 6ヶ月以上やり取りのないアカウントをフォロー解除

複数のフィルターを組み合わせることができます。例えば「2015年1月1日以前かつキーワード「大学」を含む投稿をすべて削除」も1つのフィルター組み合わせで実現できます。

5

スタートをクリックして実行する

青い「クリーンアップ開始」ボタンをクリックします。拡張機能は該当する各項目を開き、人間の速度(1項目あたり約0.9秒)で削除 / いいね解除 / フォロー解除ボタンをクリックし、進捗カウンターをリアルタイムで更新します。

次のように使えます:

  • サイドパネルでリアルタイムに動作を見守る
  • タブを最小化して通常通りブラウザを使う
  • ラップトップを完全に閉じる — ブラウザの再起動をまたいで状態が保持されるので、戻ったときに最後に完了した項目から再開します
  • 休憩したいときはいつでも一時停止
  • 実行中に気が変わったら停止

典型的な1,000件のライブラリなら、実行全体で約15分かかります。3,000件なら、約45分のバックグラウンド時間を見込んでください — 放置していても問題ありません。

一括削除を正確にする4つのフィルター

フィルターは鈍器の刀をメスに変えます。Facebook用SocialEraserで利用可能な4つと、それぞれが解決するユースケースを以下に示します。

日付範囲

特定の日付の前後で、投稿をまとめて削除します。最もよく使われるフィルターです。最近の活動を残して古い時代を消すのに使います。リブランドや面接の前にクリーンなFacebookが欲しい場合にも、「より前」の方向がぴったりです。

キャプションキーワード

テキストに特定の単語を含む投稿だけを削除します。1つのトピックを一掃するのに最適 — ブランド案件、過激な投稿、ファンダムの時代、その他ライブラリには手を加えず一部分だけ消したい場合に便利です。

投稿タイプ

ステータスのみ、写真投稿のみ、リンクシェアのみ、またはチェックインのみに絞り込みます。他のノイズを消す前に特定の投稿形式を残したい場合や、写真の思い出を守る場合に便利です。

友達非活動

Nヶ月以上やり取りのないアカウントをフォロー解除します。6ヶ月で中程度のクリーンアップ、24ヶ月で徹底的なクリーンアップを行います。本当の友達を残しつつ、10年前のソーシャルネットワークの肥大化を一掃する最良のフィルターです。

4つのいずれも自由に組み合わせられます。最も人気があるのは「2015年1月1日以前かつ友達24ヶ月以上非活動」の組み合わせ — 最近の過去を残し、深いアーカイブを一掃します。

削除されるものと残されるもの

SocialEraserは指示した対象にだけ手を加えます。具体的には:

  • タイムライン投稿にチェックON — 自分のステータス、写真投稿、リンクシェア、チェックインが(モードに応じて)1件ずつ削除またはアーカイブされます。
  • 写真にチェックON — アップロードしたすべての写真とアルバムが削除されます。
  • いいねにチェックON — いいねしたすべての投稿とページのいいねが解除されます。
  • 保存アイテムにチェックON — コレクションに保存したすべての投稿が保存解除されます。
  • 友達にチェックON — 友達リストにあるすべてのアカウントが、やり取りの頻度順にフォロー解除されます。
  • コメントにチェックON — 投稿したすべてのコメントが削除されます。
  • メッセージにチェックON — すべてのMessengerスレッドが削除されます(片側のみ — 受信者には残ります)。
  • 管理ページにチェックON — 自分が管理するページのすべての投稿が削除されます。

SocialEraserが触らないもの: アカウント設定、プロフィール写真、カバー写真、自己紹介、ユーザー名、友達リスト、グループ、イベント、Marketplace出品、明示的にチェックを入れなかったFacebook上のその他のコンテンツ。

安全性、レート制限、アカウント保護

SocialEraserは人間の速度で通常のクリックをシミュレートします — API呼び出しも、スクレイピングも、ヘッドレスブラウザも、大量自動化もありません。拡張機能はあなたがクリックするのと同じボタンを1つずつ、間に遅延を入れてクリックするだけです。平均は1項目あたり約0.9秒で、不自然に見えないようランダムな揺らぎを持たせています。

ただし、他の第三者ツールと同様に共通の前提があります。Meta Platforms, Inc.とは関係がありません。一括ツールの使用は自己責任で行ってください。FacebookのUIが変更されても、当社のリモートコンフィグ更新システムが数時間以内に新しいセレクタスキーマを配信します — 拡張機能の再インストールは不要です。

レート制限に達した場合、拡張機能は自動的に一時停止し、「再接続中」インジケーターを表示し、制限が解除されると再開します。ブラウザの再起動をまたいで状態が保持されるため、3,000件の実行を複数日に分割しても、重複や漏れは発生しません。

繰り返しになりますが: 一括実行の前にFacebookのデータアーカイブをリクエストしてください。公式の「情報をダウンロード」ページが、削除後に投稿を取り戻す唯一の手段です。さらに安全性を高めるため、最初の実行ではアーカイブオプション(完全削除ではなく)を使用することをお勧めします — アーカイブされた投稿はアクティビティログからいつでも復元できます。

よくある質問

すべてのFacebook投稿を削除するのにどのくらい時間がかかりますか?

SocialEraserは1項目あたり約0.9秒で処理します。1,000件の投稿で約15分、3,000件なら約45分のバックグラウンド時間です。寝ている間、仕事中、その他の活动中も実行したままにしておけます。

特定の日付以前の投稿だけ削除できますか?

はい。サイドパネルの開始日フィルターを使用します。その日付以降に投稿された投稿は保持され、それより古いものはすべて削除されます。これは最も一般的なフィルターで、リブランドや面接の前にすべてを消すために使う人が多いです。

やり取りのないアカウントを一括フォロー解除できますか?

はい。友達非活動フィルターで閾値を設定します。24ヶ月以上やり取りのない友達はフォロー解除され、それ以外はそのまま残ります。一般的な選択肢は24ヶ月です — 本当の友達を残しつつ、10年前のソーシャルネットワークの肥大化を一掃します。

写真やいいね、コメントの削除にも対応していますか?

はい。SocialEraserは写真の一括削除、投稿の一括いいね解除、保存アイテムのクリア、友達の一括フォロー解除、コメントの一括削除が可能です。同じフィルターシステム(日付、キーワード、投稿タイプ、友達非活動)がすべてのクリーンアップ種別に対応します。消したい対象に合致するチェックボックスを入れてください。

アカウントが停止されますか?

SocialEraserは人間の速度で通常のクリックをシミュレートし、Facebookのレート制限を尊重します。他の第三者ツールと同様の注意書きが適用されます — 私たちはMeta Platforms, Inc.とは関係がなく、一括ツールの使用は自己責任で行ってください。

削除後に元に戻すことはできますか?

完全な削除は復元できません。最初の実行ではアーカイブオプション(完全削除ではなく)を使用することを強くお勧めします。これによりアクティビティログから気が変わった項目をレビューして復元できます。アーカイブがより安全なデフォルトです。

Facebookのモバイルアプリでも動作しますか?

いいえ。Facebookのモバイルアプリはブラウザ拡張機能に対応していません。SocialEraserはデスクトップのChromeで動作します。専用のモバイルコンパニオンは2027年のロードマップに含まれています。

数千件の投稿がある場合はどうなりますか?

SocialEraserは大きなライブラリも問題なく処理します。レート制限に達した場合は自動で一時停止し、同じ場所から再開できます。ブラウザの再起動をまたいで状態が保持されるため、3,000件のライブラリを複数日に分割しても、重複や漏れは発生しません。

Facebookをきれいにしませんか?

Facebook用SocialEraserは発売前です。ウェイトリストに登録して、配信時にいち早くアクセスしてください。
