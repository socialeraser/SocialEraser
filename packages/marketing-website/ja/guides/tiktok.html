TikTok動画を一括削除する方法(ステップバイステップガイド)

なぜTikTok動画を一括削除するのか?

ほとんどの人は、いつかクリーンなTikTokが欲しくなる瞬間があります — リブランド、転職、新しい街への引っ越し、失恋後のやり直し、あるいは「いいね」タブが自分の2017年のプロフィールだと気付く瞬間です。問題は、TikTokの公式の「動画を削除」ワークフローは1動画ずつしか行えないことです。非表示は数タップで済みますが、一括削除機能は搭載されていません。

200件なら午後のタップで済みます。2,000件なら1週間。20,000件なら、何かしらの助けなしには事実上不可能です。

そこで活躍するのがTikTok用SocialEraserです。ブラウザで動作し、人間の速度で通常のクリックをシミュレートし、日付・再生回数・キャプションキーワード・コンテンツ種別でフィルターできます。ルールを設定し、スタートをクリックして、あとは離れるだけ。このガイドではその全工程を解説します。

始める前に: データをバックアップする

このステップだけは少し手間がかかるので、最初の実行前に必ず行ってください。TikTokの動画を一度削除したら復元できません — ごみ箱も、元に戻すボタンも、30日間の復元期間もありません。TikTokは削除を永続的なものと扱います。

削除した動画を(たとえ削除後でも)取り戻す唯一の公式な方法は、始める前にTikTokから「データのダウンロード」アーカイブをリクエストすることです。アーカイブは24〜48時間以内にダウンロード可能なZIPとして届き、すべての動画、いいね、コメント、フォロワーが個別のファイルとして含まれています。

  1. ブラウザで tiktok.com を開き、ログインします。
  2. 設定 → アカウント → データのダウンロード(または直接 tiktok.com/setting/download-your-data)に移動します。
  3. JSON + MP4 形式を選び、データをリクエストをクリックします。
  4. TikTokからダウンロードリンクがメール届くのを待ちます。通常は数時間、遅くとも48時間です。

ヒント: すべてを削除する予定でも、最初にこのアーカイブをリクエストしてください。無料で、あなた自身のデータであり、もし気が変わったときに戻る手段を残してくれます。

ステップバイステップ: すべてのTikTok動画を一括削除する

全体の流れはセットアップに約60秒、実際の削除はバックグラウンドで実行されます。

1

TikTok用SocialEraserをインストールする

TikTok用SocialEraserのページを開き、無料でインストールをクリックします。Chromeウェブストアが開くので、Chromeに追加をクリックしてインストールを確定してください。拡張機能のアイコンがツールバーに表示されます。

アカウントも、サインアップも、メールアドレス認証も不要です。拡張機能は無料で、チップで支えられています — Chrome 120+ と Edge 120+ で動作します。

2

tiktok.comを開いてログインする

SocialEraserをインストールしたブラウザで tiktok.com にアクセスし、通常通りアカウントにログインしてください。拡張機能はパスワードを扱いません — TikTok自身のログイン機構が使われます。SocialEraserは、認証後のページ上でのみ動作します。

ツールバーにあるSocialEraserアイコンをクリックしてサイドパネルを開きます。ログインが確定すると、上部のステータスインジケーターが黄色から緑色に変わります。

3

クリーンアップ対象を選ぶ

サイドパネルで、削除したい対象をチェックします。1つだけ選んでも、組み合わせても構いません:

  • あなたの動画 — あなたがアップロードしたTikTok動画
  • リポスト — 他のクリエイターからリポストした動画
  • いいね — いいねタブにあるすべての動画
  • お気に入り — 非公開の「お気に入り」コレクション
  • フォロー中 — あなたがフォローしているアカウント

1種類(例えばアップロードした動画だけ)だけをクリーンアップしたい場合は、そのチェックボックスだけをオンにします。拡張機能は他の対象を完全にスキップします。

4

フィルターを設定する(任意だが推奨)

フィルターこそが、精密な一括削除と、すべてを更地にする荒っぽい削除を分けるポイントです。4つのフィルターについては次のセクションで詳しく解説しますが、最も一般的な設定は以下のとおりです:

  • 開始日: 最近の動画は残し、特定の日付以前のものをすべて削除
  • 最低再生回数: 一定以上の再生回数の動画は残し、それ以下を削除
  • キーワード: キャプションに特定のワードを含む動画だけを削除
  • コンテンツタイプ: オリジナル、リポスト、またはデュエット/ステッチのみを対象

複数のフィルターを組み合わせることができます。例えば「2023年1月1日以前かつ1,000再生未満の動画をすべて削除」も1つのフィルター組み合わせで実現できます。

5

スタートをクリックして実行する

ピンクからシアンへの「クリーンアップ開始」ボタンを押します。拡張機能は該当する各項目を開き、人間の速度(1項目あたり約0.9秒)で削除 / いいね解除 / フォロー解除ボタンをクリックし、進捗カウンターをリアルタイムで更新します。

次のように使えます:

  • サイドパネルでリアルタイムに動作を見守る
  • タブを最小化して通常通りブラウザを使う
  • ラップトップを完全に閉じる — ブラウザの再起動をまたいで状態が保持されるので、戻ったときに最後に完了した項目から再開します
  • 休憩したいときはいつでも一時停止
  • 実行中に気が変わったら停止

典型的な1,000件のライブラリなら、実行全体で約15分かかります。20,000件なら、4〜5時間のバックグラウンド時間を見込んでください — 放置していても問題ありません。

一括削除を正確にする4つのフィルター

フィルターは鈍器の刀をメスに変えます。TikTok用SocialEraserで利用可能な4つと、それぞれが解決するユースケースを以下に示します。

日付範囲

特定の日付の前後で、動画をまとめて削除します。最もよく使われるフィルターです。最近の作品を残して古い時代を消すのに使います。リブランドや面接の前にクリーンなTikTokが欲しい場合にも、「より前」の方向がぴったりです。

再生回数下限

再生回数の閾値以上の動画を残し、それ以外を削除します。1,000に設定してブレイクした瞬間を残し、0にすれば再生回数に関わらずすべてを削除します。バズった動画を残しつつ、再生0回の孤独な動画を一掃するのに最適なフィルターです。

キャプションキーワード

キャプションに特定のワードやハッシュタグを含む動画だけを削除します。1つのトピックを消すのに最適 — ブランド案件、過激な投稿、ファンダムの時代、その他ライブラリには手を加えず一部分だけ消したい場合に便利です。

コンテンツタイプ

オリジナル、リポスト、またはデュエット/ステッチのみを対象にします。シェアを消す前にクリエイティブな作品を保護したい場合や、特定のコラボレーションを残して他をすべて消したい場合に便利です。

4つのいずれも自由に組み合わせられます。最も人気があるのは「2023年1月1日以前かつ1,000再生未満」の組み合わせ — ブレイクした時代を残し、実験的な動画を一掃します。

削除されるものと残されるもの

SocialEraserは指示した対象にだけ手を加えます。具体的には:

  • あなたの動画にチェックON — あなたがアップロードしたTikTok動画(公開・非公開・限定公開)が1件ずつ削除されます。
  • リポストにチェックON — リポストがプロフィールから削除されます。注: TikTokには「リポストの取り消し」ボタンが別途ありません。リポストを削除すると、その動画はプロフィール表示からも消えます。
  • いいねにチェックON — いいねタブにあるすべての動画のいいねが解除されます。
  • お気に入りにチェックON — 非公開のお気に入りコレクションにあるすべての動画が削除されます。
  • フォロー中にチェックON — フォローしているすべてのアカウントが順番にフォロー解除されます。途中で再開したい場合は順序が重要ですが、そうでない場合は効果は同じです。

SocialEraserが触らないもの: アカウント設定、DM、他の人の動画へのコメント、下書き、プロフィール写真、自己紹介、ユーザー名、その他明示的にチェックを入れなかったTikTok上のあらゆるコンテンツ。

安全性、レート制限、アカウント保護

SocialEraserは人間の速度で通常のクリックをシミュレートします — API呼び出しも、スクレイピングも、ヘッドレスブラウザも、大量自動化もありません。拡張機能はあなたがクリックするのと同じボタンを1つずつ、間に遅延を入れてクリックするだけです。平均は1項目あたり約0.9秒で、不自然に見えないようランダムな揺らぎを持たせています。

ただし、他の第三者ツールと同様に共通の前提があります。ByteDance Ltd.とは関係がなく、一括ツールの使用は自己責任で行ってください。TikTokのUIが変更されても、当社のリモートコンフィグ更新システムが数時間以内に新しいセレクタスキーマを配信します — 拡張機能の再インストールは不要です。

レート制限に達した場合、拡張機能は自動的に一時停止し、「再接続中」インジケーターを表示し、制限が解除されると再開します。ブラウザの再起動をまたいで状態が保持されるため、20,000件の実行を複数日に分割しても、重複や漏れは発生しません。

繰り返しになりますが: 一括実行の前にTikTokのデータアーカイブをリクエストしてください。公式の「データのダウンロード」ページが、削除後に動画を取り戻す唯一の手段です。

よくある質問

すべてのTikTok動画を削除するのにどのくらい時間がかかりますか?

SocialEraserは1項目あたり約0.9秒で処理します。1,000件の動画で約15分、20,000件なら約4〜5時間のバックグラウンド時間です。寝ている間、仕事中、その他の活动中も実行したままにしておけます。

特定の日付以前の動画だけ削除できますか?

はい。サイドパネルの開始日フィルターを使用します。その日付以降に投稿された動画は保持され、それより古いものはすべて削除されます。これは最も一般的なフィルターで、リブランドや面接の前にすべてを消すために使う人が多いです。

バズった動画を残して他を削除できますか?

はい。最低再生回数フィルターで閾値を設定します。閾値を超える動画は残り、それ以下の動画は削除されます。一般的な選択肢は1,000再生です — ブレイクした瞬間を残し、再生0回の動画を一掃します。

リポスト、いいね、フォローも削除できますか?

はい。SocialEraserはリポストの一括削除、動画の一括いいね解除、非公開のお気に入りのクリア、フォローの一括解除が可能です。同じフィルターシステム(日付、再生回数、キーワード、コンテンツタイプ)が5つのクリーンアップ種別すべてに対応します。消したい対象に合致するチェックボックスを入れてください。

一括削除でアカウントが停止されますか?

SocialEraserは人間の速度で通常のクリックをシミュレートし、TikTokのレート制限を尊重します。第三者ツール全般に言える注意書きと同様 — 私たちはByteDance Ltd.とは関係がなく、一括ツールの使用は自己責任で行ってください。

削除後に元に戻すことはできますか?

いいえ。動画が一度削除されると、数秒以内にTikTokから消えます。本ガイドの最初のステップがTikTokの公式アーカイブを使ったバックアップであるのはこのためです。

TikTokのモバイルアプリでも動作しますか?

いいえ。TikTokのモバイルアプリはブラウザ拡張機能に対応していません。SocialEraserはデスクトップのChromeで動作します。専用のモバイルコンパニオンは2027年のロードマップに含まれています。

数万件の動画がある場合はどうなりますか?

SocialEraserは大きなライブラリも問題なく処理します。レート制限に達した場合は自動で一時停止し、同じ場所から再開できます。ブラウザの再起動をまたいで状態が保持されるため、20,000件のライブラリを複数日に分割しても、重複や漏れは発生しません。

TikTokをきれいにしませんか?

無料Chrome拡張機能。アカウント不要、サインアップ不要。100%ローカルで、データはブラウザから一切出ません。
