清理できるもの

YouTube の 8 種類の清理を 1 つの拡張機能で

視聴履歴からコメントの墓場まで — 1 つだけ選んでも、組み合わせても、すべてを連続で実行することもできます。

視聴履歴

ワンクリックで視聴履歴をすべて清理、または日付範囲で絞り込み。アルゴリズムを今のあなたの興味に合わせてリセットします。

検索履歴

YouTube で入力したすべての検索ワードを削除。「深夜 3 時に蛇口の直し方を調べた」履歴もプライベートに保てます。

いいねした動画

「高く評価した動画」プレイリストのすべての動画からいいねを解除。よくある用途：思春期の「黒歴史」を消すこと。

保存した再生リスト

「後で見る」などのプライベート再生リストや他のカスタム再生リストを削除。任意の再生リストから個別の動画を一括で除外できます。

チャンネル登録

チャンネル登録を一括解除。6 ヶ月以上活動のないチャンネルを見つけてワンクリックで整理できます。

自分のコメント

あらゆる動画に投稿した自分のコメントをすべて削除。同じエンジン、別のタイムライン。

コミュニティ投稿

自分のチャンネルの YouTube コミュニティタブから投稿を一括削除します。

自分のアップロード動画

アップロードした自分の動画を一括削除。リダイレクトする前にサブチャンネルを整理するのに便利です。

Chrome ウェブストアから無料で。アカウント登録、サインアップ、支払い不要。2 回クリックでインストール完了。

youtube.com を開き、普段通りログイン。SocialEraser のサイドパネルアイコンをクリックし、清理する対象とフィルター（日付範囲、活動期間）を選択します。

SocialEraser が人間と同じ速度でクリックをシミュレート。ライブカウンターを確認するか、ラップトップを閉じて翌日に戻ってきてください。状態は保持されます。

フィルター

盲目的にではなく、的確に

一括清理の価値はスピードです。ただし精度も同様に重要。フィルターを組み合わせて、不要な項目だけを正確に削除しましょう。

日付範囲

任意の日付より前または後の履歴を清理。最近のチュートリアルは残し、2012 年の恥ずかしい履歴を一掃できます。最もよく使われるフィルターです。

チャンネルの活動状況

チャンネル登録を整理する際に、6 ヶ月以上投稿のないチャンネルを検出。ワンクリックで一括登録解除。活動期間はお好みで調整できます。

タイトルキーワード

キーワードに一致する履歴やいいねだけを清理。転職活動の時期、議論になったトピック、特定のファン活動など、特定のテーマだけを一掃するのに最適です。

動画の長さ

再生時間でフィルター。短尺の自動再生セッションだけを消し、本当に見た長尺コンテンツは残すといった使い方ができます。

FAQ

YouTube 固有の質問

清理の範囲

視聴履歴をすべて清理できますか？

はい。ワンクリックで視聴履歴のすべてのエントリを清理できます。日付による絞り込み（例：1 年以上前）も可能で、最近の動画は残して他を清理できます。1 件あたり約 0.9 秒で処理 — 1,000 件の動画なら約 15 分で完了します。

履歴を清理するとおすすめに影響しますか？

はい — それが本来の目的でもあります。視聴履歴を清理すると YouTube のアルゴリズムが白紙状態に戻り、おすすめが数年前に見たものではなく、今のあなたの興味を反映するようになります。

活動していないチャンネルを一括登録解除できますか？

はい。活動期間（例：6 ヶ月、1 年）を設定すると、SocialEraser がその期間内に投稿のないチャンネルを検出し、一括で登録解除します。期間はお好みで調整可能です。

再生リストはどうなりますか？

自分が所有する再生リストを丸ごと削除したり、個別の動画を除去したりできます。SocialEraser は他のユーザーの再生リストを編集することはできません。

安全性

自分のチャンネルに影響しますか？

視聴履歴、いいね、チャンネル登録を清理しても、影響を受けるのはあなたの個人アカウントのみです — あなたのチャンネル、投稿、購読者数はそのまま維持されます。第三者ツール全般に言える注意点として、当社は Google LLC とは一切関係ありません。

清理後に元に戻せますか？

YouTube は、清理された視聴履歴や検索履歴のバックアップを保存しません。一括清理を行う前に、Google Takeout で YouTube データをリクエストし、個人アーカイブとして保存しておくことを強くおすすめします。

YouTube の UI が変更された場合は？

SocialEraser は DOM セレクタースキーマをリモート CDN から取得しています。YouTube が UI を変更した際、当社側でアップデートを配信するため、拡張機能の再インストールは不要です。アップデートは自動的かつバックグラウンドで行われます。

技術面

新しい YouTube のレイアウトに対応していますか？

はい。SocialEraser は新しい YouTube マテリアルデザインとレガシーレイアウトの両方に対応しています。セレクタースキーマは毎週更新され、YouTube の最新の UI 変更に追従します。

YouTube モバイルアプリでも動作しますか？

いいえ。YouTube のモバイルアプリではブラウザ拡張機能を使用できません。SocialEraser はデスクトップの Chrome 上で動作します。専用のモバイルコンパニオンアプリは 2027 年のロードマップに含まれています。

複数の Google アカウントで使えますか？

はい — Google アカウントごとに別のブラウザプロフィールを使用してください。1 つのプロフィール内での複数アカウント対応はロードマップに含まれています。
