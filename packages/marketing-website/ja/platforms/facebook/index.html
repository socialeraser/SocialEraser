8 種類の Facebook クリーンアップを 1 つの拡張機能で

タイムラインの投稿から友達リストまで — 1 つだけ選んでも、組み合わせて実行しても、すべて連続実行しても OK。

タイムライン投稿

ステータス更新を一括削除またはアーカイブ。日付範囲やキーワードで絞り込み可能。両モードに対応。

写真とアルバム

写真とフォトアルバムを一括削除。タグ付け写真・未タグ写真のどちらも対応。先にバックアップを。

いいねとリアクション

数千件の投稿のいいね・リアクションを取り消し。アクティビティログはあなたの公開プロフィール — 整理しましょう。

保存したアイテム

保存した投稿とリンクを整理。「あとで読む」のはずが 1,200 件溜まってしまったあなたに。

友達

友達を一括解除。交流頻度で絞り込んで、5 年以上連絡を取っていない 600 人の友達を見つけ出せます。

あなたのコメント

他の人の投稿に書き込んだコメントをすべて削除。同じエンジン、別タイムラインで動作します。

メッセージ

Messenger スレッドを一括削除。受信箱ゼロを目指す方や、長年放置されていたチャットをリセットしたい方に。

管理中のページ

管理している Facebook ページの投稿を一括削除。休眠状態のブランドページの整理に便利です。

Eraser for Facebook をインストール

Chrome ウェブストアから無料で入手。アカウント、サインアップ、支払い一切不要。2 回クリックで完了します。

facebook.com を開いてログイン

facebook.com を開いて通常通りログイン。SocialEraser のサイドパネルアイコンをクリック。クリーンアップ対象とフィルター（日付範囲、キーワード、交流頻度）を選択。

スタートをクリック。放置して OK。

SocialEraser が人間の速度でクリックを再現。カウンターの様子を見るもよし、ノートパソコンを閉じて翌日に戻るもよし。状態は保持されます。

フィルター

一括でも、精密に — 闇雲に消しません

一括削除の醍醐味はスピードですが、精度も同じくらい重要です。フィルターを組み合わせて、本当に消したいものだけを削除できます。

日付範囲

任意の日付より前または後の投稿を削除 / アーカイブ。最近の投稿は残して、2008〜2012 年の投稿を一掃。最もよく使われるフィルターです。

キーワード一致

特定の単語やフレーズを含む投稿だけを削除。転職活動中、激しい議論、ファンダム活動など、特定のトピックをまとめて消去するのに便利。

投稿タイプ

ステータス更新のみ、写真のみ、共有リンクのみ、チェックインのみに絞り込み。1 つの形式を残して他を消去する場合に便利です。

友達の非アクティブ期間

友達解除時、何か月 / 何年の期間を設定するか指定。その期間内にやり取りがなかった人を全員友達解除。デフォルトは 24 か月。

Facebook 固有の質問

クリーンアップの範囲

古い Facebook の投稿を一括削除できますか？

はい。日付フィルター（例：2015 年 1 月 1 日より前）またはキーワード一致を設定すれば、SocialEraser が条件に一致するすべての投稿を削除またはアーカイブします。1 件あたり約 0.9 秒、1,000 件の投稿で約 15 分です。

削除とアーカイブのどちらですか？

両方に対応しています。SocialEraser は 2 つのモードをサポート：完全削除（投稿が永久に消える）とアーカイブ（タイムラインには表示されないがアクティビティログから復元可能）。アーカイブの方が安全なデフォルトです。

友達を一括で解除できますか？

はい。交流頻度で友達を絞り込み、6 か月以上やり取りしていない人を全員友達解除できます。好みに応じて調整可能。推奨デフォルトは 24 か月です。

他人が投稿した自分の写真はどうなりますか？

SocialEraser は自分がタグ付けされた写真をタイムラインから非表示にできますが、他の人がアップロードした写真を削除することはできません。完全に削除するには、タグを外して投稿者に削除を依頼してください。

安全性

アカウントが BAN されますか？

SocialEraser は人間の速度で通常のクリックを再現し、速率制限を守ります。第三者ツール全般に言える注意点として、Meta Platforms, Inc. とは関係がなく、一括ツールの利用は自己責任で行ってください。

削除を取り消せますか？

完全削除は復元できません。最初の実行にはアーカイブモードの使用を強くおすすめします。気が変わったものはアクティビティログから確認・復元できます。

Facebook の UI が変わったらどうなりますか？

SocialEraser は DOM セレクタースキーマをリモート CDN から取得しています。Facebook が UI を変更した際、こちらからアップデートをプッシュするため拡張機能の再インストールは不要です。自動でバックグラウンドで更新されます。

技術的なこと

新しい Facebook のレイアウトでも動作しますか？

はい。SocialEraser は最新の facebook.com レイアウトと旧 FB5 表示の両方に対応。セレクタースキーマは毎週更新され、Facebook の最新の UI 変更に追従します。

Facebook のモバイルアプリでも動作しますか？

いいえ。Facebook モバイルアプリはブラウザ拡張機能を許可していないため、SocialEraser はデスクトップの Chrome 上で動作します。専用のモバイル版は 2027 年のロードマップに含まれています。

複数の Facebook アカウントで使えますか？

はい。アカウントごとに別のブラウザプロファイルを使用してください。1 つのプロファイルで複数アカウント対応は今後のロードマップにあります。
