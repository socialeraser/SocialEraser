削除できる内容

6 種類の X 整理機能を 1 つの拡張機能で

よく使われるものから忘れがちなものまで。1つだけ選んでも、組み合わせても、連続実行することもできます。

オリジナルツイート

自分のオリジナル投稿を一括削除。日付範囲やキーワードで絞り込み可能。バックアップとしてまず JSON にエクスポートできます。

リプライ

リプライ履歴を一括消去。転職活動やリブランド、心機一転のタイミングに便利です。

リツイート

リポストを一括取り消し。元の投稿者のツイートはそのまま残し、自分のプロフィールから共有を削除します。

いいね

何百、何千もの投稿への「いいね」を解除。SocialEraser の中で最も利用されている機能です。

ブックマーク

保存したツイートを消去。「あとで読む」コレクションの整理に。

フォロー

リスト、相互フォローなし、最終ツイート日で指定してフォロー解除。休眠アカウントをワンクリックで発掘できます。

使い方

3 ステップで完了

Eraser for X をインストール

Chrome ウェブストアから無料で入手。アカウント登録、サインアップ、支払いは不要。2 回クリックするだけでツールバーにピン留めされ、100% ブラウザ内で動作します。

x.com を開いてログイン

Chrome で x.com を開き、普段どおりログイン。SocialEraser のサイドパネルアイコンをクリックし、整理対象（ツイート、リプライ、リツイート、いいね、ブックマーク、フォロー）にチェックを入れ、フィルター条件（日付範囲、キーワード、エンゲージメント下限）を設定します。

開始をクリックして、あとは放置

SocialEraser が人の速度でクリックを再現。ライブカウンターを横目で確認しても、ノートパソコンを閉じて翌日戻ってきても大丈夫。ブラウザの再起動をまたいで状態が保持されます。いつでも一時停止可能。

フィルター

雑に消すのではなく、的確に整理する

一括削除の意義はスピードですが、精度も同様に重要です。フィルターを組み合わせて、不要なものだけを的確に削除できます。

日付範囲

任意の日付より前または後に投稿されたツイートを削除。コロナ禍の思い出は残して、2014 年の黒歴史は消したい、そんな時に。最もよく使われるフィルターです。

キーワード一致

特定の単語やフレーズを含むツイートだけを削除。転職活動中、議論中、特定のファンダムなど、トピックをまたいだ整理に便利です。

メディア種別

画像付き、動画付き、リンク付き、テキストのみなど、メディア種別で絞り込み可能。投稿を消す前にメディアをバックアップしておきたい時にも便利です。

エンゲージメント下限

一定数以上のいいねやリプライが付いたツイートを残す設定。誰にも反応されなかった投稿を一掃します。0 に設定すれば、すべてが削除対象に。

よくある質問

X 特有の質問

整理の範囲

古いツイートを一括削除できますか？

はい。日付フィルター（例：2020年1月1日より前）を設定すれば、条件に一致するすべてのツイートを SocialEraser が削除します。1項目あたり約 0.9 秒で動作し、1,000 件のツイートなら約 15 分で完了します。

リプライだけ削除できますか？

はい。投稿タイプフィルターを使えば、リプライのみ（あるいはオリジナル投稿のみ、リツイートのみ）を対象にできます。日付やキーワードのフィルターと組み合わせられます。

いいねが多いツイートは残せますか？

はい。エンゲージメント下限フィルターで、最小いいね数を設定できます。閾値より上のツイートは保持され、下回るものは削除されます。

スレッドはどうなりますか？

SocialEraser はツイートを1件ずつ削除します。スレッド内のツイートだった場合、返信チェーンは孤児状態になりますが、表示自体は残ります。スレッドを完全に削除したい場合は、日付フィルターを使って新しい順（逆時系列順）に削除してください。

安全性

アカウントは凍結されますか？

SocialEraser は通常のクリックを人の速度で再現し、API 制限（レートリミット）を尊重します。他のサードパーティ製ツールと同様のご注意点はありますが、当社は X Corp. と提携関係にあるわけではなく、一括削除ツールのご使用はお客様の自己責任となります。

削除を取り消せますか？

いいえ。一度削除されたツイートは、数秒以内に X の公開表示から消えます。初めて一括削除を行う際は、必ず事前にエクスポート（アーカイブ保存）機能をご利用になることを強くおすすめします。

X の UI が変更された場合はどうなりますか？

SocialEraser は DOM セレクタのスキーマをリモート CDN から取得しています。X が UI を変更した際、当社側でアップデートを配信するため、拡張機能の再インストールは不要です。アップデートは自動的かつサイレントに適用されます。

技術面

新しい x.com のレイアウトでも動作しますか？

はい。SocialEraser は x.com と旧 twitter.com ドメインの両方に対応しています。セレクタスキーマは X の最新の UI 変更に合わせて毎週更新されます。

モバイル（モバイルブラウザの x.com）でも動作しますか？

Android 版 Chrome ではほとんどの機能が動作します。iOS Safari は Apple の拡張機能モデルの制約により制限が多く、iOS 専用アプリは 2027 年のロードマップに含まれています。

複数の X アカウントで実行できますか？

はい — アカウントごとに別のブラウザプロフィールをご利用ください。1つのプロフィール内で複数アカウントを切り替える機能は、ロードマップに含まれています。
