プライバシー ポリシー (Privacy Policy)

最終更新日: 2026 年 6 月 20 日  ·  発効日: 即時

TL;DR: SocialEraser はブラウザまたはモバイル アプリ内で 100% ローカルに動作します。 お客様のデータを収集、送信、保存、売却することはありません。サーバーは存在しません。 拡張機能をアンインストールすると、保存されたすべての設定が完全に削除されます。

1. 概要 (Overview)

SocialEraser (以下「当社」) は、ソーシャル メディア プラットフォーム (現在 X / Twitter、将来的に TikTok、YouTube、Facebook に対応) 上の投稿、いいね、フォロー、保存済みアイテムを一括削除するのに役立つブラウザ拡張機能およびモバイル アプリケーションを提供します。 本プライバシー ポリシーは、拡張機能が取り扱うデータとその方法を説明します。

2. 当社が収集しないデータ (Data We Do NOT Collect)

  • ❌ ユーザー名、パスワード、ログイン資格情報は 収集しません。
  • ❌ 投稿、いいね、フォロー、保存済みアイテムは 収集しません。
  • ❌ IP アドレス、ブラウザ フィンガープリント、デバイス識別子は 収集しません。
  • ❌ 分析、追跡ピクセル、サードパーティ スクリプトは 使用しません。
  • ❌ いかなるデータもサードパーティに販売または共有することは ありません。
  • ❌ バックエンド サーバーは 一切運営していません。侵害されるべきものすら存在しません。

3. 端末にローカル保存されるデータ (Data Stored Locally)

拡張機能は chrome.storage.local を使用して、以下の非機密性の高い設定のみを保存します。すべてお客様の端末上に保持されます。

  • 言語設定 — サポートされる 8 言語の中から選択された言語。
  • 日次使用カウンター — 当日実行したクリーンアップ操作の回数 (使用量分析およびレート制限検出のため)。
  • 投げ銭レシート (該当する場合) — プロジェクトをサポートする際に、最終的な寄付確認のローカル キャッシュ。
  • リモート設定キャッシュ — 最新のプラットフォーム セレクター設定 (例：「削除」ボタンの CSS セレクター) のコピー。24 時間ごとに更新されます。

これらのデータがお客様の端末から外部に送信されることはありません。Chrome の chrome://extensions → SocialEraser → 「service worker」→ Storage から確認できます。拡張機能をアンインストールすると、保存されたすべてのデータが完全に削除されます。

4. 権限の説明 (Permissions Explained)

以下の権限は Chrome ブラウザ拡張機能のみ に適用されます。 SocialEraser の Web サイト (本ページ) および計画中の Android / iOS アプリは、ブラウザ レベルの権限を一切要求しません。

拡張機能は、以下の Chrome 権限を要求します。それぞれの理由は以下のとおりです:

権限必要な理由
storage設定をローカルに保存するため (セクション 3 参照)。
activeTab「Start Cleanup」をクリックしたときに現在の X.com ページを読み取るため。
sidePanelChrome のサイド パネルにクリーンアップ UI を表示するため。
tabs/likes, /bookmarks, /following, /with_replies タブ間を移動して各セクションをクリーンアップするため。
scriptingx.com ページにクリーンアップ エンジンを挿入するため。
<all_urls>ユーザーのアクティブ タブに対して chrome.scripting.executeScript() を呼び出すために、バックグラウンド サービス ワーカーに必要なホスト アクセスを付与します。コンテンツ スクリプト自体はマニフェストの content_scripts.matches フィールドによって x.com / twitter.com に限定されており、他のドメインでは実行されません。

5. 拡張機能の仕組み (How the Extension Works)

  1. Chrome で SocialEraser アイコンをクリックすると、サイド パネルが開きます。
  2. クリーンアップする項目 (投稿 / いいね / フォロー / 保存済みアイテム) を選択します。
  3. Start Cleanup をクリックします。
  4. x.com 内で動作する拡張機能のコンテンツ スクリプトが、「削除」「いいね解除」「ブックマーク解除」「フォロー解除」ボタン (お客様が手動でクリックするのと同じボタン) のクリックをシミュレートします。
  5. すべてのクリックはお客様のブラウザ内で、お客様の端末で実行されます。データが送信されることはありません。

6. サードパーティ サービス (Third-Party Services)

拡張機能は、サードパーティの分析、広告、エラー追跡、テレメトリ サービスとは 一切連携していません。Google Analytics、Sentry、Mixpanel、Plausible、または類似のツールは使用していません。

7. リモート設定 (Remote Configuration)

拡張機能の更新を強制することなく X.com の UI 変更に対応するため、拡張機能は socialeraser.app から小さな JSON ファイル (セレクター設定) を定期的に取得します。この取得は匿名で行われ (ユーザー識別子は送信されません)、個人データを含みません。ネットワーク接続を無効にすることで、リモート設定の更新を無効にできます。その場合、拡張機能はバンドルされたフォールバック セレクターを使用します。

8. 児童 (Children)

本拡張機能は 13 歳未満の児童を対象としていません。児童から故意にデータを収集することはありません。児童が拡張機能を使用したと思われる場合は、ご連絡ください。対応を検討します。

9. 国際ユーザー (GDPR / CCPA / LGPD)

データを一切収集しないため、GDPR / CCPA / LGPD のデータ主体の権利 (アクセス、削除、ポータビリティ) は自明のうちに満たされます: アクセス、削除、書き出すべきデータが存在しません。ご懸念がある場合は、下記の連絡先までお問い合わせください。

10. 本ポリシーの変更 (Changes to This Policy)

本プライバシー ポリシーを更新する場合があります。変更は新しい「最終更新日」とともにこのページに掲載されます。変更が重要な場合は、拡張機能のリリース ノートおよび (次回のアップデートの際) サイド パネル UI 内に記載します。

11. お問い合わせ (Contact)

ご質問、ご懸念、ご要望は support@socialeraser.app までメールでお送りください。7 営業日以内にご返信します。

12. オープン ソース (Open Source)

拡張機能のソース コードは github.com/socialeraser/SocialEraser で公開されています。コードを監査し、自分でビルドし、プライバシーに関する主張を検証していただくことを歓迎します。
