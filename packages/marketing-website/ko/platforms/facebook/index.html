정리 범위

Facebook 정리 8종을 한 확장으로

타임라인 게시글부터 친구 목록까지 — 하나만 실행하거나, 섞거나, 연속으로 모두 실행하세요.

타임라인 게시글

상태 업데이트를 일괄 삭제하거나 보관하세요. 날짜 범위나 키워드로 필터링하세요. 두 가지 모드 모두 지원됩니다.

사진 및 앨범

사진과 사진 앨범을 일괄 삭제하세요. 태그된 사진과 태그되지 않은 사진 모두 지원됩니다. 먼저 백업하세요.

좋아요 및 리액션

수천 개의 게시글의 좋아요를 취소하거나 리액션을 되돌리세요. 활동 로그는 당신의 공개 기록입니다 — 깨끗이 정리하세요.

저장 항목

저장한 게시글과 링크를 정리하세요. 1,200개로 깊어진 "나중에 읽을게요" 더미를 비우세요.

친구

친구를 일괄 삭제하세요. 상호작용 빈도로 필터링하여 5년 이상 연락하지 않은 600명의 친구를 찾으세요.

내 댓글

다른 사람의 게시글에 남긴 모든 댓글을 삭제하세요. 같은 엔진, 다른 타임라인.

메시지

메신저 스레드를 일괄 삭제하세요. 받은편지함 0 또는 수 년간 잠자던 채팅의 깨끗한 리셋에 유용합니다.

관리하는 페이지

관리하는 Facebook 페이지의 게시글을 일괄 삭제하세요. 휴면 상태인 브랜드 페이지를 정리할 때 유용합니다.

날짜 범위

특정 날짜 이전 또는 이후의 게시글을 삭제하거나 보관하세요. 최근 인생 업데이트는 남기고 2008~2012년 시기는 정리하세요. 가장 많이 사용되는 필터입니다.

키워드 일치

특정 단어나 구문이 포함된 게시글만 삭제하세요. 주제를 정리할 때 좋습니다 — 구직기, 격한争论, 덕질 시절 등.

게시글 유형

상태 업데이트만, 사진만, 공유 링크만, 또는 체크인만 필터링하세요. 다른 형식을 정리하면서 한 형식을 유지할 때 유용합니다.

친구 휴면 기간

FAQ

Facebook 관련 질문

정리 범위

오래된 Facebook 게시글을 일괄 삭제할 수 있나요?

네. 날짜 필터(예: 2015년 1월 1일 이전) 또는 키워드 일치를 설정하면 SocialEraser가 일치하는 모든 게시글을 삭제하거나 보관합니다. 항목당 약 0.9초로 실행되며, 게시글 1,000개에 약 15분이 걸립니다.

삭제인가요 보관인가요?

둘 다 가능합니다. SocialEraser는 두 가지 모드를 지원합니다: 영구 삭제(게시글이 영원히 사라짐)와 보관(타임라인에서는 숨겨지지만 활동 로그에서 복구 가능). 보관이 더 안전한 기본값입니다.

계정을 일괄 친구 삭제할 수 있나요?

네. 상호작용 빈도로 친구를 필터링하고 6개월 이상 연락하지 않은 모든 친구를 삭제하세요. 취향에 따라 조정 가능합니다. 권장 기본값: 24개월.

다른 사람이 올린 내 사진은 어떻게 되나요?

SocialEraser는 내가 태그된 사진을 타임라인에서 숨길 수 있지만, 다른 사람이 업로드한 사진은 삭제할 수 없습니다. 완전한 제거를 위해서는 태그를 해제하고 업로드한 사람에게 삭제를 요청하세요.

안전성

제 계정이 정지될 수 있나요?

SocialEraser는 사람의 속도로 정상적인 클릭을 시뮬레이션하며 속도 제한을 준수합니다. 모든 제3자 도구와 동일한 주의 사항이 적용됩니다 — 당사는 Meta Platforms, Inc.와 제휴 관계가 없으며, 일괄 도구 사용은 본인의 판단에 따릅니다.

삭제를 취소할 수 있나요?

영구 삭제는 복구할 수 없습니다. 첫 실행에는 보관 모드를 사용하는 것을 강력히 권장합니다. 그래야 마음이 바뀌면 활동 로그에서 검토하고 복구할 수 있습니다.

Facebook이 UI를 변경하면 어떻게 되나요?

SocialEraser는 원격 CDN에서 DOM 선택기 스키마를 가져옵니다. Facebook이 UI를 변경하면 업데이트를 푸시하므로 확장 프로그램을 다시 설치할 필요가 없습니다. 업데이트는 자동으로 조용히 진행됩니다.

기술

새 Facebook 레이아웃에서도 작동하나요?

네. SocialEraser는 최신 facebook.com 레이아웃과 이전 FB5 보기를 모두 지원합니다. 선택기 스키마는 매주 업데이트되어 Facebook의 최신 UI 변경 사항에 맞춥니다.

Facebook 모바일 앱에서도 작동하나요?

아니요. Facebook 모바일 앱은 브라우저 확장을 허용하지 않습니다. SocialEraser는 데스크톱 Chrome에서 실행됩니다. 전용 모바일 동반 앱은 2027년 로드맵에 있습니다.

여러 Facebook 계정에서 실행할 수 있나요?

네 — 각 계정에 대해 별도의 브라우저 프로필을 사용하세요. 단일 프로필 내 다중 계정 기능은 로드맵에 있습니다.
