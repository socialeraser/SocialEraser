기본

SocialEraser는 정확히 무엇을 하나요?

원하는 소셜 미디어 플랫폼에서 게시물, 좋아요, 팔로우, 저장 항목 등을 일괄 삭제합니다. 정리할 항목을 선택하고 필터를 설정하면, 일시정지/재개 상태 머신과 함께 브라우저에서 한 항목씩 처리합니다.

SocialEraser는 정말 무료인가요?

네 — SocialEraser는 팁으로 운영됩니다. 무료 버전이 완전한 버전이며, 제한도 없고 계정도 없고 체험 기간도 없습니다. 팁은 환영이지만 절대 의무가 아닙니다. 유료 벽도, 할당량도, 업그레이드 알림도 없습니다.

계정을 만들어야 하나요?

아닙니다. 계정 0, 가입 0, 이메일 0. 설치하고 사용하면 끝입니다.

어떤 브라우저를 지원하나요?

현재는 Chrome과 Edge입니다. Firefox와 Safari는 2026년 로드맵에 있습니다.

정리하는 동안 브라우저를 열어두어야 하나요?

정리가 실행되는 브라우저 탭은 열어 두어야 하지만 최소화할 수 있습니다. Chrome을 닫으면 실행이 일시정지되며, 다시 열면 멈춘 지점부터 정확히 이어서 진행됩니다. 사이드 패널은 탭에서 분리해 항상 보이게 할 수 있습니다.

개인정보 보호 및 보안

비밀번호를 요구하나요?

절대 없습니다. SocialEraser는 소셜 미디어 비밀번호를 절대 묻지 않습니다. 이미 로그인된 브라우저 세션 안에서 동작합니다. 볼 수 있는 자격 증명이 없기 때문에 귀하의 계정을 볼 수 없습니다.

내 데이터를 수집하나요?

아닙니다. 데이터를 받는 서버가 없습니다. 확장 프로그램은 전적으로 클라이언트 측에서 실행됩니다. chrome.storage에 저장되는 유일한 것은 환경설정(언어, 일일 카운터)이며, 클릭 한 번으로 지울 수 있습니다.

설치해도 안전한가요?

확장 프로그램은 Chrome Web Store에 등록되기 전 Google의 검토를 거칩니다. 소스 코드는 GitHub에서 오픈소스로 공개되어 있어 누구나 감사할 수 있습니다(MIT 라이선스). 모든 원격 스크립트를 차단하는 엄격한 콘텐츠 보안 정책을 사용합니다.

사용으로 계정이 정지될 수 있나요?

SocialEraser는 사람의 속도로 정상적인 클릭을 시뮬레이션합니다. 플랫폼에는 속도 제한이 있으며, 저희는 이를 존중합니다. 다만 어떤 플랫폼과도 제휴 관계가 아니므로 일괄 도구 사용의 판단은 본인에게 달려 있습니다. 이는 모든 제3자 도구에 동일하게 적용됩니다.

확장 프로그램이 서버와 통신하지 않는지 확인할 수 있나요?

네. 확장 프로그램이 실행되는 동안 Chrome DevTools → Network 탭을 여세요. 제3자 도메인으로 나가는 요청은 0건임을 확인할 수 있습니다. 발생하는 네트워크 활동은 브라우저와 정리 중인 소셜 플랫폼 사이의 것뿐 — 수동으로 할 때와 동일한 요청입니다. 전체 소스 코드는 누구나 감사할 수 있도록 GitHub에 공개되어 있습니다.

작동 방식

일괄 삭제는 실제로 어떻게 작동하나요?

확장 프로그램은 플랫폼의 원래 인터페이스(예: x.com)를 열고, 각 항목의 작업 메뉴를 찾아, 삭제를 클릭하고, 확인을 클릭합니다. 사용자가 수동으로 하는 것과 동일 — 다만 더 빠르고 지치지 않습니다.

어떤 필터를 사용할 수 있나요?

날짜 범위(X 이전/이후의 게시물 삭제), 키워드("Y"가 포함된 게시물만 삭제), 미디어 유형(이미지만, 텍스트만, 영상만), 계정(특정 핸들만 삭제). 필터 조합은 OR이 아닌 AND로 작동합니다.

실행을 일시정지하고 재개할 수 있나요?

네. 브라우저를 닫은 후에도 일시정지/재개는 언제든 가능합니다. 상태는 chrome.storage.local에 저장되므로 재시작 후에도 유지됩니다. 다음 날 돌아와서 멈춘 지점부터 이어서 진행하세요.

속도는 어느 정도인가요?

평균 항목당 1초 미만. 1,000개 항목 실행은 약 15분이 걸립니다. 위의 시간 계산기로 정확한 예상 시간을 확인하세요.

삭제 전에 내보낼 수 있나요?

네. X 확장 프로그램에는 삭제 전에 각 항목의 archive.json을 저장하는 "파일 내보내기" 옵션이 있습니다. 다른 플랫폼에도 출시와 동시에 동일한 옵션을 추가하고 있습니다.

정리 중 인터넷이 끊기면 어떻게 되나요?

확장 프로그램이 연결 끊김을 감지하고 실행을 일시정지하며 "재연결 중..." 표시기를 보여줍니다. 네트워크가 복구되면 마지막 완료 항목부터 이어서 진행합니다. 어디에도 전송되거나 저장되지 않습니다 — 재개 상태는 브라우저의 로컬 저장소에만 존재합니다.

정리를 취소할 수 있나요?

아닙니다 — 게시물이 플랫폼에서 삭제되면 사라집니다. 정리 실행 전 (가능한 경우) "파일 내보내기" 옵션으로 JSON 백업을 저장하시길 권장합니다. 그러면 삭제된 내용의 기록을 남길 수 있습니다.

vs. 대안 도구

SocialEraser vs TweetDelete?

TweetDelete는 X API를 사용하며 OAuth 인증을 요구합니다. 서버 측에서 동작하므로 데이터가 기기를 떠납니다. SocialEraser는 브라우저에서 동작하며, 자격 증명을 절대 묻지 않고, X만이 아닌 5개 플랫폼에서 작동합니다.

SocialEraser vs Semiphemeral?

Semiphemeral은 기간 기반 자동 삭제 — X개월 이상 된 트윗을 일정에 따라 삭제합니다. SocialEraser는 주문형 일괄 삭제 — 정리할 항목, 시기, 양을 직접 선택하세요. 다른 작업을 위한 다른 도구입니다.

SocialEraser vs Redact?

Redact는 API로 트윗을 일괄 삭제하는 유료 개발자 도구입니다. API 키와 개발자 계정이 필요합니다. SocialEraser는 무료이며, API 접근이 필요 없고, X 외의 플랫폼에서도 작동합니다.

SocialEraser vs 수동 작업?

1,000개 항목을 수동으로 삭제하려면 약 3시간의 연속 클릭이 필요합니다. SocialEraser는 같은 작업을 약 15분 만에 처리하며, 지치지 않고, 확인 대화상자를 놓치지 않고, 새벽 2시에 잘못된 버튼을 누르지 않습니다. 같은 작업을 자동화한 것입니다.

플랫폼별

YouTube / Instagram / Facebook은 언제 출시되나요?

X와 TikTok은 이미 사용 가능합니다. YouTube, Instagram, Facebook은 고유한 UI 패턴, 속도 제한, 안티 봇 조치가 있어, 한 번에 한 플랫폼씩 출시하여 안정적일 때까지 조정한 뒤 다음으로 넘어갑니다.

모바일 앱(iOS / Android)도 무료인가요?

네 — Android와 iOS 앱도 데스크톱과 동일한 팁 운영 모델을 따릅니다. 제한도, 계정도, 유료 벽도 없습니다. 무료 버전이 완전한 버전입니다.

여러 계정에서 SocialEraser를 사용할 수 있나요?

Chrome에서는 각 계정마다 별도의 브라우저 프로필을 사용할 수 있습니다. 단일 프로필에서 여러 계정 지원은 로드맵에 있습니다.

먼저 플랫폼에 로그인해야 하나요?

네 — SocialEraser는 이미 로그인된 브라우저 세션 안에서 동작합니다. 로그인을 대신 처리하지 않고, 쿠키를 저장하지 않으며, 인증 토큰을 건드리지 않습니다. 세션이 만료되면 다시 로그인하고 실행을 이어가세요.
