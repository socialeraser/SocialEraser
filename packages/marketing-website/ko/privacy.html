개인정보 보호 정책 (Privacy Policy)

최종 업데이트: 2026년 6월 20일  ·  즉시 발효

TL;DR: SocialEraser는 브라우저 또는 모바일 앱에서 100% 로컬로 실행됩니다. 당사는 사용자의 데이터를 수집, 전송, 저장 또는 판매하지 않습니다. 서버가 없습니다. 확장 프로그램을 제거하면 저장된 모든 환경설정이 영구적으로 제거됩니다.

1. 개요 (Overview)

SocialEraser("we", "us", "our")는 소셜 미디어 플랫폼(현재 X / Twitter; 향후 TikTok, YouTube, Facebook 지원)에서 게시물, 좋아요, 팔로우, 저장 항목을 일괄 삭제하는 데 도움이 되는 브라우저 확장 프로그램 및 모바일 애플리케이션을 제공합니다. 본 개인정보 보호 정책은 확장 프로그램이 처리하는 데이터와 그 처리 방식을 설명합니다.

2. 당사가 수집하지 않는 데이터 (Data We Do NOT Collect)

  • ❌ 당사는 사용자의 사용자 이름, 비밀번호 또는 로그인 자격 증명을 수집하지 않습니다.
  • ❌ 당사는 사용자의 게시물, 좋아요, 팔로우 또는 저장 항목을 수집하지 않습니다.
  • ❌ 당사는 사용자의 IP 주소, 브라우저 핑거프린트 또는 기기 식별자를 수집하지 않습니다.
  • ❌ 당사는 분석, 추적 픽셀 또는 제3자 스크립트를 사용하지 않습니다.
  • ❌ 당사는 제3자와 일체의 데이터를 판매하거나 공유하지 않습니다.
  • ❌ 당사는 일체의 백엔드 서버를 운영하지 않습니다. 침해할 것이 없습니다.

3. 로컬에 저장되는 데이터 (사용자 기기에) (Data Stored Locally)

확장 프로그램은 chrome.storage.local을 사용하여 다음의 비민감 환경설정만 저장하며, 모두 사용자 기기에 남아 있습니다.

  • 언어 환경설정 — 선택한 8개 지원 언어 중 하나.
  • 일일 사용 카운터 — 오늘 수행한 정리 작업 수(사용 분석 및 속도 제한 감지용).
  • 팁 단지 영수증 (해당되는 경우) — 프로젝트를 지원하기로 선택한 경우 마지막 기부 확인의 로컬 캐시.
  • 원격 구성 캐시 — 최신 플랫폼 셀렉터 구성(예: "삭제" 버튼에 대한 CSS 셀렉터)의 사본, 24시간마다 새로고침됩니다.

이 데이터 중 어느 것도 사용자 기기를 떠나지 않습니다. Chrome의 chrome://extensions → SocialEraser → "service worker" → Storage를 통해 검사할 수 있습니다. 확장 프로그램을 제거하면 모든 저장된 데이터가 영구적으로 삭제됩니다.

4. 권한 설명 (Permissions Explained)

아래 권한은 Chrome 브라우저 확장 프로그램에만 적용됩니다. SocialEraser 웹사이트(본 페이지)와 계획된 Android / iOS 앱은 브라우저 수준의 권한을 요청하지 않습니다.

확장 프로그램은 다음 Chrome 권한을 요청하며, 각각 아래에 정당화됩니다.

권한 (Permission)필요한 이유 (Why we need it)
storage환경설정을 로컬에 저장(섹션 3 참조).
activeTab"정리 시작"을 클릭할 때 현재 X.com 페이지를 읽습니다.
sidePanelChrome의 사이드 패널에 정리 UI를 표시합니다.
tabs각 섹션을 정리하기 위해 /likes, /bookmarks, /following, /with_replies 탭 사이를 탐색합니다.
scriptingx.com 페이지에 정리 엔진을 주입합니다.
<all_urls>백그라운드 서비스 워커가 사용자의 활성 탭에 대해 chrome.scripting.executeScript()를 호출하는 데 필요한 호스트 액세스를 부여합니다. 콘텐츠 스크립트 자체는 매니페스트의 content_scripts.matches 필드에 의해 x.com / twitter.com으로 제한됩니다 — 확장 프로그램은 다른 도메인에서 실행되지 않습니다.

5. 확장 프로그램 작동 방식 (How the Extension Works)

  1. Chrome에서 SocialEraser 아이콘을 클릭하면 사이드 패널이 열립니다.
  2. 정리할 항목(게시물 / 좋아요 / 팔로우 / 저장 항목)을 선택합니다.
  3. 정리 시작을 클릭합니다.
  4. x.com 내부에서 실행되는 확장 프로그램의 콘텐츠 스크립트가 "삭제" / "좋아요 취소" / "북마크 제거" / "언팔로우" 버튼 — 사용자가 수동으로 클릭하는 것과 동일한 버튼 — 에 사용자 클릭을 시뮬레이션합니다.
  5. 모든 클릭은 사용자 브라우저의 사용자 기기에서 발생합니다. 데이터는 어디로도 전송되지 않습니다.

6. 제3자 서비스 (Third-Party Services)

확장 프로그램은 일체의 제3자 분석, 광고, 오류 추적 또는 원격 측정 서비스와 통합되지 않습니다. 당사는 Google Analytics, Sentry, Mixpanel, Plausible 또는 유사한 도구를 사용하지 않습니다.

7. 원격 구성 (Remote Configuration)

확장 프로그램을 강제로 업데이트하지 않고 X.com UI 변경을 처리하기 위해, 확장 프로그램은 socialeraser.app에서 작은 JSON 파일(셀렉터 구성)을 주기적으로 가져옵니다. 이 가져오기는 익명이며(사용자 식별자가 전송되지 않음) 개인 데이터를 포함하지 않습니다. 네트워크 연결을 비활성화하여 원격 구성 새로고침을 비활성화할 수 있습니다; 확장 프로그램은 번들된 대체 셀렉터를 사용합니다.

8. 어린이 (Children)

확장 프로그램은 13세 미만의 아동을 대상으로 하지 않습니다. 당사는 아동으로부터 의도적으로 데이터를 수집하지 않습니다. 어린이가 확장 프로그램을 사용한 것으로 판단되면, 우려 사항을 해결할 수 있도록 당사에 연락해 주십시오.

9. 국제 사용자 (GDPR / CCPA / LGPD)

당사가 데이터를 수집하지 않기 때문에 GDPR / CCPA / LGPD 데이터 주체 권리 (액세스, 삭제, 이동성)는 사소하게 충족됩니다: 액세스, 삭제 또는 내보낼 것이 없습니다. 우려 사항이 있으시면 아래 주소로 연락주십시오.

10. 본 정책의 변경 (Changes to This Policy)

당사는 본 개인정보 보호 정책을 업데이트할 수 있습니다. 변경 사항은 새 "최종 업데이트" 날짜와 함께 본 페이지에 게시됩니다. 중요한 변경 사항이 있는 경우 확장 프로그램의 릴리스 노트와 (다음 업데이트의 경우) 사이드 패널 UI 안에 메모를 남기겠습니다.

11. 연락처 (Contact)

질문, 우려 또는 요청 사항이 있으신가요? support@socialeraser.app로 이메일을 보내주십시오. 영업일 기준 7일 이내에 답변드립니다.

12. 오픈소스 (Open Source)

확장 프로그램의 소스 코드는 github.com/socialeraser/SocialEraser에서 확인할 수 있습니다. 코드를 감사하고, 직접 빌드하고, 당사의 개인정보 보호 주장을 검증하시기를 권장합니다.
