Facebook 게시물 일괄 삭제하는 방법 (단계별 가이드 2026)

2012년의 타임라인이 다시 떠오르는 것이 지치시나요? 구직, 리브랜딩 또는 새로운 시작 전에 깔끔한 Facebook을 원하시나요? 이 가이드는 계정의 모든 게시물, 사진, 좋아요, 댓글, 친구를 안전하게 필터와 함께 1,000개당 약 15분 만에 일괄 삭제하는 과정을 안내합니다.

왜 Facebook 게시물을 일괄 삭제하나요?

대부분의 사람들은 어느 시점에서 깔끔한 Facebook을 원하게 됩니다. 리브랜딩, 구직, 도시 이동, 새로운 시작, 또는 타임라인이 2012년의 본인 박물관이라는 사실을 깨닫게 되는 경우. 문제는: Facebook의 공식 "게시물 삭제" 워크플로는 한 번에 한 게시물씩입니다. 내장된 일괄 삭제, 일괄 작업, 키워드 필터, 기간별 정리 기능이 없습니다.

200개의 게시물의 경우 오후 내내 클릭해야 합니다. 2,000개의 경우 일주일이 걸립니다. 3,000개의 경우 도움 없이는 사실상 불가능합니다.

이것이 바로 Facebook용 SocialEraser가 필요한 이유입니다. 브라우저에서 실행되며, 사람의 속도로 정상적인 클릭을 시뮬레이션하고 날짜, 키워드, 게시물 유형 또는 친구 비활성별로 필터링할 수 있습니다. 규칙을 설정하고 시작을 클릭한 다음 자리를 비웁니다. 이 가이드는 전체 과정을 안내합니다.

시작하기 전: 데이터 백업하기

이것이 가장 중요한 단계이며 첫 번째 실행 전에 수행해야 합니다. Facebook 게시물을 영구적으로 삭제하면 사라집니다. 휴지통도, 실행 취소도 없습니다. Facebook "아카이브" 기능은 복구 가능한 상태를 제공하지만 인앱 아카이브 버튼을 사용해야 합니다.

게시물을 복구하는 공식적인 방법(삭제 후에도)은 시작 전에 Facebook에서 정보 다운로드 아카이브를 요청하는 것입니다. 아카이브는 모든 게시물, 사진, 댓글, 좋아요, 친구, 메시지가 독립 실행형 파일로 포함됩니다.

  1. 브라우저에서 facebook.com을 열고 로그인합니다.
  2. 직접 facebook.com/dyi로 이동합니다.
  3. JSON + HTML 형식을 선택하고 사용 가능한 모든 정보를 선택한 다음 다운로드 요청을 클릭합니다.
  4. Facebook이 다운로드 링크를 이메일로 보낼 때까지 기다립니다. 보통 몇 시간이 걸리지만 때로는 최대 며칠이 걸립니다.

팁: 모든 것을 삭제할 계획이더라도 먼저 이 아카이브를 요청하세요. 무료이며, 사용자의 데이터이며, 마음이 바뀌었을 때 복구할 수 있는 방법을 제공합니다. 보너스로, 첫 번째 실행에서는 인앱 아카이브 옵션(영구 삭제 대신)을 사용할 수도 있습니다. 아카이브된 게시물은 작업 로그에서 언제든지 복원할 수 있습니다.

단계별: 모든 Facebook 게시물을 일괄 삭제

전체 과정은 설정에 약 60초가 걸립니다. 실제 삭제는 백그라운드에서 실행됩니다.

1

Facebook용 SocialEraser 설치

Facebook용 SocialEraser 페이지로 이동하여 Facebook 사전 등록을 클릭하세요. 확장 프로그램은 현재 사전 출시 중이며 — 사전 등록자는 출시 시 조기 접근 권한을 받습니다. (또한 오늘 완전히 출시된 X용 SocialEraser를 설치할 수도 있습니다.)

2

facebook.com을 열고 로그인

SocialEraser를 설치한 브라우저에서 facebook.com으로 이동합니다. 평소처럼 계정에 로그인하세요. 확장 프로그램은 비밀번호를 처리하지 않습니다. Facebook 자체의 로그인이 처리합니다. SocialEraser는 이미 인증된 후에만 페이지에서 작동합니다.

툴바에서 SocialEraser 아이콘을 클릭하여 사이드 패널을 엽니다. 상단의 상태 표시기가 노란색에서 녹색으로 바뀌면 로그인이 확인된 것입니다.

3

정리할 항목 선택

사이드 패널에서 삭제할 항목의 확인란을 선택하세요. 하나만 선택하거나 결합할 수 있습니다.

  • 타임라인 게시물 — 본인의 상태 업데이트, 사진 게시물, 링크 공유, 체크인
  • 사진 — 업로드한 사진과 사진 앨범
  • 좋아요 — 좋아요를 누른 모든 게시물과 페이지
  • 저장 항목 — 비공개 저장 게시물과 링크
  • 친구 — 친구로 추가한 계정
  • 댓글 — 다른 게시물에 작성한 본인 댓글
  • 메시지 — 시작한 메신저 스레드
  • 관리하는 페이지 — 관리하는 페이지의 게시물

한 유형(예: 타임라인 게시물만)만 정리하려면 그것만 선택하세요. 확장 프로그램은 다른 항목은 완전히 건너뜁니다.

4

필터 설정 (선택 사항이지만 권장)

필터는 정밀한 일괄 삭제와 모두 삭제하기 사이의 차이입니다. 네 가지 필터는 다음 섹션에서 자세히 다루지만 가장 일반적인 설정은 다음과 같습니다.

  • 시작 날짜: 최근 게시물은 유지하고 특정 날짜 이전의 모든 것을 삭제
  • 키워드: 캡션에 특정 단어가 포함된 게시물만 삭제
  • 게시물 유형: 상태만, 사진만, 링크만, 체크인만
  • 친구 비활성: 6개월 이상 대화하지 않은 계정을 친구 삭제

여러 필터를 결합할 수 있습니다. 예를 들어 "2015년 1월 1일 이전, '대학교' 키워드 포함된 모든 게시물 삭제"는 단일 필터 조합입니다.

5

시작을 클릭하고 실행되도록 두기

파란색 정리 시작 버튼을 누르세요. 확장 프로그램이 일치하는 각 항목을 열고 사람의 속도(항목당 약 0.9초)로 삭제/좋아요 취소/친구 삭제 버튼을 클릭하며 실시간 진행률 카운터를 업데이트합니다.

다음을 수행할 수 있습니다.

  • 사이드 패널에서 실시간으로 작동하는 것을 보기
  • 탭을 최소화하고 평소처럼 브라우저 사용
  • 노트북을 완전히 닫기 — 상태는 브라우저 재시작 시에도 유지되므로 돌아오면 마지막으로 완료된 항목에서 재개됩니다
  • 언제든지 일시 중지를 눌러 휴식
  • 실행 중 마음이 바뀌면 중지를 누르세요

일반적인 1,000개 게시물 라이브러리의 경우 전체 실행에는 약 15분이 걸립니다. 3,000개의 경우 45분의 백그라운드 시간이 예상되며 관리가 필요하지 않습니다.

정밀한 일괄 삭제를 가능하게 하는 네 가지 필터

필터는 무딘 도구를 메스로 바꿉니다. 다음은 Facebook용 SocialEraser에서 사용 가능한 네 가지 필터와 각 필터가 해결하는 사용 사례입니다.

날짜 범위

특정 날짜 이전 또는 이후에 게시된 게시물을 삭제합니다. 이것이 가장 일반적인 필터입니다. 최근 활동을 유지하고 이전 시대를 정리하는 데 사용하세요. 리브랜딩이나 면접 전에 깔끔한 Facebook을 원하는 경우에도 "이전" 방향이 적합한 도구입니다.

캡션 키워드

텍스트에 특정 단어가 포함된 게시물만 삭제합니다. 라이브러리의 나머지를 건드리지 않고 단일 주제를 정리하는 데 좋습니다. 브랜드 계약, 격한 의견, 팬덤 시대 등.

게시물 유형

상태만, 사진 게시물만, 링크 공유만, 체크인만 삭제하도록 필터링합니다. 다른 형식을 정리하면서 특정 게시물 형식을 유지하거나, 상태 잡음을 지우기 전에 사진 추억을 백업하는 데 유용합니다.

친구 비활성

N개월 동안 상호작용하지 않은 계정을 친구 삭제합니다. 중간 정리를 위해 6개월, 깊은 정리를 위해 24개월로 설정하세요. 이것은 실제 친구를 유지하면서 10년 전의 소셜 네트워크 부담을 정리하는 최고의 필터입니다.

네 가지 중 어느 것이든 결합할 수 있습니다. 가장 인기 있는 조합은 "2015년 1월 1일 이전 AND 친구 24개월+ 비활성" — 최근 과거는 유지하고 깊은 아카이브는 정리합니다.

삭제되는 것과 유지되는 것

SocialEraser는 지시한 것만 건드립니다. 구체적으로:

  • 타임라인 게시물 확인란 ON — 본인의 상태 업데이트, 사진 게시물, 링크 공유, 체크인이 (모드에 따라) 삭제되거나 아카이브됩니다.
  • 사진 확인란 ON — 업로드한 모든 사진과 앨범이 삭제됩니다.
  • 좋아요 확인란 ON — 좋아요를 누른 모든 게시물과 페이지의 좋아요가 취소됩니다.
  • 저장 항목 확인란 ON — 모음집의 모든 저장 게시물의 저장이 취소됩니다.
  • 친구 확인란 ON — 친구 목록의 모든 계정이 상호작용 빈도 순서대로 친구 삭제됩니다.
  • 댓글 확인란 ON — 작성한 모든 댓글이 삭제됩니다.
  • 메시지 확인란 ON — 모든 메신저 스레드가 삭제됩니다(일방적; 수신자는 여전히 사본을 가지고 있습니다).
  • 관리하는 페이지 확인란 ON — 관리하는 페이지의 모든 게시물이 삭제됩니다.

SocialEraser가 건드리지 않는 것: 계정 설정, 프로필 사진, 커버 사진, 자기소개, 사용자 이름, 친구 목록, 그룹, 이벤트, 마켓플레이스 목록 또는 명시적으로 확인란을 선택하지 않은 Facebook의 다른 콘텐츠.

안전성, 속도 제한, 계정 안전

SocialEraser는 사람의 속도로 정상적인 클릭을 시뮬레이션합니다 — API 호출 없음, 스크래핑 없음, 헤드리스 브라우저 없음, 대량 자동화 없음. 확장 프로그램은 문자 그대로 동일한 버튼을 한 번에 하나씩 클릭하며 각 클릭 사이에 지연이 있습니다. 평균은 항목당 약 0.9초이며, 로봇처럼 보이지 않도록 무작위 지터가 있습니다.

그렇긴 하지만 모든 타사 도구와 마찬가지로 동일한 주의 사항이 적용됩니다. 우리는 Meta Platforms, Inc.와 제휴하지 않습니다. 대량 도구 사용은 본인의 재량에 따릅니다. Facebook이 UI를 변경하면 원격 구성 업데이트 시스템이 몇 시간 내에 새 선택자 스키마를 푸시합니다 — 확장 프로그램 재설치가 필요하지 않습니다.

속도 제한에 도달하면 확장 프로그램이 자동으로 일시 중지되고 "재연결 중" 표시기를 표시하며 제한이 해제되면 재개됩니다. 상태는 브라우저 재시작 시에도 유지되므로 3,000개 게시물 실행을 중복이나 건너뛴 항목 없이 여러 날에 걸쳐 분할할 수 있습니다.

한 번 더: 첫 번째 대량 실행 전에 Facebook 데이터 아카이브를 요청하세요. 공식 "정보 다운로드" 페이지가 삭제 후 게시물을 복구할 수 있는 유일한 방법입니다. 안전을 추가하려면 첫 번째 실행에서 아카이브 옵션(영구 삭제 대신)을 사용하는 것이 좋습니다. 아카이브된 게시물은 작업 로그에서 언제든지 복원할 수 있습니다.

자주 묻는 질문

내 모든 Facebook 게시물을 삭제하는 데 얼마나 걸리나요?

SocialEraser는 항목당 약 0.9초를 처리합니다. 1,000개의 게시물은 대략 15분이 걸립니다. 3,000개의 게시물은 약 45분의 백그라운드 시간이 걸립니다. 잠을 자거나, 일하거나, 다른 일을 하는 동안 실행되도록 둘 수 있습니다.

특정 날짜 이전에 게시된 게시물만 삭제할 수 있나요?

예. 사이드 패널의 시작 날짜 필터를 사용하세요. 해당 날짜 이후에 게시된 게시물은 유지됩니다. 이전의 모든 것은 삭제됩니다. 이것이 가장 일반적인 필터이며 많은 사용자가 리브랜딩이나 면접 전에 모든 것을 정리하기 위해 사용합니다.

대화하지 않는 계정을 일괄 친구 삭제할 수 있나요?

예. 친구 비활성 필터를 사용하고 임계값을 설정하세요. 24개월 이상 상호작용하지 않은 친구는 친구 삭제되고 다른 모든 사람은 유지됩니다. 일반적인 선택은 24개월입니다 — 실제 친구를 유지하고 10년 전의 소셜 네트워크 부담을 정리합니다.

사진, 좋아요, 댓글 삭제에도 작동하나요?

예. SocialEraser는 사진 일괄 삭제, 게시물 일괄 좋아요 취소, 저장 항목 정리, 계정 일괄 친구 삭제, 댓글 일괄 삭제를 수행할 수 있습니다. 동일한 필터 시스템(날짜, 키워드, 게시물 유형, 친구 비활성)이 모든 정리 유형에 적용됩니다. 사라지게 하려는 항목에 해당하는 확인란을 선택하세요.

대량 삭제하면 계정이 정지되나요?

SocialEraser는 사람의 속도로 정상적인 클릭을 시뮬레이션하며 Facebook의 속도 제한을 존중합니다. 모든 타사 도구와 마찬가지로 동일한 주의 사항이 적용됩니다. 우리는 Meta Platforms, Inc.와 제휴하지 않으며 대량 도구 사용은 본인의 재량에 따릅니다.

삭제 후 실행 취소할 수 있나요?

영구 삭제는 복구할 수 없습니다. 첫 번째 실행에서는 작업 로그에서 검토하고 마음이 바뀐 항목을 복구할 수 있도록 아카이브 옵션(영구 삭제 대신)을 사용하는 것을 강력히 권장합니다. 아카이브가 더 안전한 기본값입니다.

Facebook 모바일 앱에서도 작동하나요?

아니요. Facebook 모바일 앱은 브라우저 확장을 허용하지 않습니다. SocialEraser는 데스크톱의 Chrome에서 실행됩니다. 전용 모바일 동반 앱은 2027년 로드맵에 있습니다.

수천 개의 게시물이 있으면 어떻게 되나요?

SocialEraser는 대규모 라이브러리도 잘 처리합니다. 확장 프로그램은 속도 제한에 도달하면 자동으로 일시 중지되며 동일한 지점에서 재개할 수 있습니다. 상태는 브라우저 재시작 시에도 유지되므로 3,000개 게시물 라이브러리를 중복이나 건너뛴 항목 없이 여러 날에 걸쳐 분할할 수 있습니다.
