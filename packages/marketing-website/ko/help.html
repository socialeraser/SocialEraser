도움말 & FAQ

최종 업데이트: 2026년 6월 20일

빠른 시작

  1. 홈페이지의 다운로드 섹션에서 SocialEraser를 설치합니다.
  2. Chrome 툴바에서 SocialEraser 아이콘을 클릭하여 사이드 패널을 엽니다.
  3. 정리할 플랫폼(X, TikTok 등)을 열고 계정에 로그인합니다.
  4. 사이드 패널에서 해당 플랫폼에 대해 정리할 항목을 선택합니다.
  5. 정리 시작을 클릭합니다. 진행률 표시줄과 로그가 실시간으로 표시됩니다.
  6. 언제든지 일시정지 / 재개 / 중지를 사용할 수 있습니다.

FAQ

SocialEraser는 안전한가요? 계정이 정지되나요?

SocialEraser는 사용자가 수동으로 수행하는 것과 동일한 작업("삭제", "좋아요 취소", "북마크 제거", "언팔로우")을 수행합니다. 작업이 실제 브라우저 이벤트이기 때문에 수동 클릭과 구별할 수 없습니다. 그러나 X / Twitter의 자동 시스템은 대량 활동을 감지할 수 있습니다. 권장 사항은 다음과 같습니다.

  • 일정한 속도로 실행하세요 — SocialEraser는 속도 제한 감지 시 자동으로 일시정지합니다.
  • 확장 프로그램을 연속해서 여러 시간 동안 실행하지 마세요.
  • 날짜 / 키워드 필터를 사용하여 작고 목표가 있는 배치로 정리하세요.

X의 이용약관 준수에 대한 모든 책임은 사용자에게 있습니다. 자세한 내용은 이용약관을 참조하세요.

왜 일부 트윗 / 좋아요는 삭제되지 않나요?

일반적인 이유는 다음과 같습니다.

  • 고정된 트윗 — 고정된 트윗은 UI에서 삭제할 수 없습니다. 먼저 고정을 해제하세요.
  • 정지된 계정의 리트윗 — UI를 통해 취소할 수 없습니다.
  • 다른 계정의 트윗 — 올바른 계정에 로그인했는지 확인하세요.
  • UI 변경 — X.com은 자주 UI를 업데이트합니다. 사이드 패널의 새로고침 아이콘을 클릭하여 최신 셀렉터 구성을 가져오세요.

DM(다이렉트 메시지)을 삭제할 수 있나요?

아닙니다. X / Twitter의 DM 삭제는 실제 네이티브 우클릭 (컨텍스트 메뉴)을 필요로 하며, 브라우저의 event.isTrusted 검사로 보호됩니다. JavaScript로 주입된 이벤트는 거부됩니다. 이를 우회하려면 chrome.debugger 권한이 필요한데, 이는 강력한 사용자 신뢰 경고를 트리거하므로 요청하지 않습니다. DM을 삭제하려면 X의 네이티브 인터페이스를 수동으로 사용하세요.

제 데이터는 어디에 저장되나요?

100% 사용자 기기에 Chrome의 로컬 저장소에 저장됩니다. 확장 프로그램은 서버가 없습니다. 확장 프로그램을 제거하면 모든 저장된 데이터가 영구적으로 삭제됩니다. 자세한 내용은 개인정보 보호 정책을 참조하세요.

어떤 언어를 지원하나요?

현재: English, 简体中文, 日本語, 한국어, Português, Español, Deutsch, Français. 사이드 패널의 국기 아이콘을 통해 전환하세요.

모바일에서도 작동하나요?

Android (Capacitor) 버전이 개발 중입니다. iOS는 향후 출시 예정입니다. support@socialeraser.app에서 대기 목록에 등록하세요.

TikTok / YouTube / Facebook은 어떻게 되나요?

곧 출시 예정입니다. 프로젝트는 활발히 개발 중입니다. 업데이트는 GitHub에서 팔로우하세요.

제거하려면 어떻게 하나요?

Chrome 툴바에서 SocialEraser 아이콘을 우클릭 → "Chrome에서 제거". 모든 로컬에 저장된 데이터가 삭제됩니다.

버그 신고

버그를 발견하셨거나 기능 요청이 있으신가요? GitHub에 이슈를 열거나 support@socialeraser.app로 이메일을 보내주세요.
