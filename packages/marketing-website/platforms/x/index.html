How It Works

Three Steps. Then It's Done.

Install SocialEraser for X

Free from the Chrome Web Store. No account, no signup, no payment. Two clicks and it's pinned to your toolbar — runs 100% in your browser.

Open x.com & log in

Go to x.com in Chrome and log in as usual. Click the SocialEraser side panel icon. Tick what to clean — tweets, replies, retweets, likes, bookmarks, follows — and set your filter (date range, keyword, engagement floor).

Click Start. Walk away.

SocialEraser simulates clicks at human speed. Watch the live counter, or close the laptop and come back tomorrow — state persists across browser restarts. Pause anytime.

Filters

Wipe Precisely — Not Blindly

The whole point of bulk is to clean fast. But precision matters too. Combine filters to delete exactly the wrong things.

Date Range

Delete tweets posted before or after any date. Keep your pandemic-era thoughts, nuke the 2014 cringe. The most common filter.

Keyword Match

Only delete tweets containing a specific word or phrase. Great for cleaning up a topic — like a job-search era, a heated debate, or a fandom.

Media Type

Filter to only delete tweets with images, only videos, only links, or only text. Useful for backing up media before wiping the post.

Engagement Floor

Keep tweets with more than X likes or replies. Wipe the lonely 0-engagement posts. Set 0 to delete everything.

FAQ

X-Specific Questions

Cleanup Scope

Can I bulk delete old tweets?

Yes. Set a date filter (e.g. before Jan 1, 2020) and SocialEraser will delete every matching tweet. Runs at ~0.9s per item — about 15 minutes for 1,000 tweets.

Can I delete only my replies?

Yes. Use the Posts type filter to target only replies (or only original posts, or only retweets). Combines with date and keyword filters.

Can I keep tweets with many likes?

Yes. Use the Engagement floor filter to set a minimum like count. Tweets above the floor are kept; below are deleted.

What about threads?

SocialEraser deletes tweets individually. If a tweet is part of a thread, the reply chain becomes orphaned but remains visible. For full thread removal, delete tweets in reverse chronological order using the date filter.

Safety

Will my account get suspended?

SocialEraser simulates normal clicks at human speed and respects rate limits. The same caveat applies as with any third-party tool — we are not affiliated with X Corp., and using bulk tools is at your own discretion.

Can I undo a delete?

No. Once a tweet is deleted, it's gone from X's public view within seconds. We strongly recommend using Export archive before any bulk run for the first time.

What if X changes their UI?

SocialEraser fetches its DOM selector schema from a remote CDN. When X ships UI changes, we push an update — no extension reinstall needed. The update is automatic and silent.

Technical

Does it work on the new x.com layout?

Yes. SocialEraser supports both x.com and legacy twitter.com domains. The selector schema is updated weekly to match X's latest UI changes.

Does it work on mobile (x.com in mobile browser)?

Chrome on Android works with most features. iOS Safari is more limited due to Apple's extension model — a dedicated iOS app is on the 2027 roadmap.

Can I run it on multiple X accounts?

Yes — use separate browser profiles for each account. Multi-account within a single profile is on the roadmap.
