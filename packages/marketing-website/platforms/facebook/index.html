Filters

Wipe Precisely — Not Blindly

The whole point of bulk is to clean fast. But precision matters too. Combine filters to delete exactly the wrong things.

Date Range

Delete or archive posts from before or after any date. Keep the recent life updates, nuke the 2008–2012 era. The most common filter.

Keyword Match

Only delete posts containing a specific word or phrase. Great for clearing a topic — like a job-search era, a heated debate, or a fandom.

Post Type

Filter to only status updates, only photos, only shared links, or only check-ins. Useful for keeping one format while wiping the rest.

Friend Inactivity

When unfriending, set a window of months/years. Unfriend everyone you haven't interacted with in that window. Default: 24 months.

FAQ

Facebook-Specific Questions

Cleanup Scope

Can I bulk delete old Facebook posts?

Yes. Set a date filter (e.g. before Jan 1, 2015) or a keyword match and SocialEraser will delete or archive every matching post. Runs at ~1.6s per item — about 25 minutes for 1,000 posts.

Does it delete or archive?

Both. SocialEraser supports two modes: permanent delete (the post is gone forever) and archive (hidden from your timeline but recoverable from the Activity Log). Archive is the safer default.

Can I bulk unfriend accounts?

Yes. Filter your friends by interaction frequency and unfriend everyone you haven't spoken to in 6+ months. Adjustable to taste. Recommended default: 24 months.

What about photos others posted of me?

SocialEraser can hide photos you're tagged in from your timeline, but it cannot delete photos that other people have uploaded. For full removal, untag and ask the uploader to remove it.

Safety

Will my account get banned?

SocialEraser simulates normal clicks at human speed and respects rate limits. The same caveat applies as with any third-party tool — we are not affiliated with Meta Platforms, Inc., and using bulk tools is at your own discretion.

Can I undo a delete?

Permanent deletes are not recoverable. We strongly recommend using archive mode for the first run, so you can review and recover anything you change your mind about from the Activity Log.

What if Facebook changes their UI?

SocialEraser fetches its DOM selector schema from a remote CDN. When Facebook ships UI changes, we push an update — no extension reinstall needed. The update is automatic and silent.

Technical

Does it work on the new Facebook layout?

Yes. SocialEraser supports both the latest facebook.com layout and the older FB5 view. The selector schema is updated weekly to match Facebook's latest UI changes.

Does it work on the Facebook mobile app?

No. The Facebook mobile app does not allow browser extensions. SocialEraser runs in Chrome on desktop. A dedicated mobile companion is on the 2027 roadmap.

Can I run it on multiple Facebook accounts?

Yes — use separate browser profiles for each account. Multi-account within a single profile is on the Pro tier roadmap.
