Filters

Wipe Precisely — Not Blindly

The whole point of bulk is to clean fast. But precision matters too. Combine filters to delete exactly the wrong things.

Date Range

Delete videos posted before or after any date. Keep your breakout hit, nuke the experimental era. The most common filter.

View Count Floor

Keep videos with more than X views. Wipe the lonely 0-view uploads, the "wait, this isn't on my main?" pile. Set 0 to delete everything.

Caption Keyword

Only delete videos with a word or hashtag in the caption. Great for clearing a topic — like a brand deal, a heated take, or a fandom era.

Content Type

Filter to only delete originals, only reposts, or only duets/stitches. Useful for backing up creative work before wiping the share.

FAQ

TikTok-Specific Questions

Cleanup Scope

Can I bulk delete old TikTok videos?

Yes. Set a date filter (e.g. before Jan 1, 2023) or a view-count floor and SocialEraser will delete every matching video. Runs at ~0.9s per item — about 15 minutes for 1,000 videos.

Can I delete only my reposts?

Yes. Use the Content type filter to target only reposts, only duets, or only original uploads. Combines with date and view-count filters.

Can I keep videos that went viral?

Yes. Use the View count floor filter to set a minimum view count. Videos above the floor are kept; below are deleted. Many users set 1,000 to preserve breakout moments.

What about duets and stitches?

SocialEraser deletes videos individually. If your video was a duet or stitch, the original creator's video remains intact — only your side of the chain is removed.

Safety

Will my account get banned?

SocialEraser simulates normal clicks at human speed and respects rate limits. The same caveat applies as with any third-party tool — we are not affiliated with ByteDance Ltd., and using bulk tools is at your own discretion.

Can I undo a delete?

No. Once a video is deleted, it's gone from TikTok's public view within seconds. We strongly recommend requesting TikTok's official Download your data archive before any bulk run for the first time.

What if TikTok changes their UI?

SocialEraser fetches its DOM selector schema from a remote CDN. When TikTok ships UI changes, we push an update — no extension reinstall needed. The update is automatic and silent.

Technical

Does it work on the new tiktok.com layout?

Yes. SocialEraser supports both tiktok.com web and the legacy desktop layout. The selector schema is updated weekly to match TikTok's latest UI changes.

Does it work on the TikTok mobile app?

No. TikTok's mobile app does not allow browser extensions. SocialEraser runs in Chrome on desktop. A dedicated mobile companion is on the 2027 roadmap.

Can I run it on multiple TikTok accounts?

Yes — use separate browser profiles for each account. Multi-account within a single profile is on the roadmap.
