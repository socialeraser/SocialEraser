FAQ

Instagram-Specific Questions

Cleanup Scope

Can I bulk delete old Instagram posts?

Yes. Set a date filter (e.g. before Jan 1, 2022) or a hashtag match and SocialEraser will delete or archive every matching post. Runs at ~1.6s per item — about 25 minutes for 1,000 posts.

Does it delete or archive?

Both. SocialEraser supports two modes: permanent delete (the post is gone forever) and archive (hidden from your profile but recoverable for 30 days in the Instagram app). Archive is the safer default.

Can I keep my most-liked posts?

Yes. Use the Engagement floor filter to set a minimum like count. Posts above the floor are kept; below are deleted or archived.

What about carousels with multiple photos?

SocialEraser deletes or archives the whole carousel as a single post. Carousel count is treated as one item in the time estimator.

Safety

Will my account get banned?

SocialEraser simulates normal taps at human speed and respects rate limits. The same caveat applies as with any third-party tool — we are not affiliated with Meta Platforms, Inc., and using bulk tools is at your own discretion.

Can I undo a delete?

Permanent deletes are not recoverable. We strongly recommend using archive mode for the first run, so you can review and recover anything you change your mind about within 30 days.

What if Instagram changes their UI?

SocialEraser fetches its DOM selector schema from a remote CDN. When Instagram ships UI changes, we push an update — no extension reinstall needed. The update is automatic and silent.

Technical

Does it work on the new Instagram layout?

Yes. SocialEraser supports both the latest instagram.com web layout and the older legacy view. The selector schema is updated weekly to match Instagram's latest UI changes.

Does it work on the Instagram mobile app?

No. The Instagram mobile app does not allow browser extensions. SocialEraser runs in Chrome on desktop. A dedicated mobile companion is on the 2027 roadmap.

Can I run it on multiple Instagram accounts?

Yes — use separate browser profiles for each account. Multi-account within a single profile is on the Pro tier roadmap.
