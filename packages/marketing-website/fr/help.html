Aide & FAQ

Dernière mise à jour : 20 juin 2026

Démarrage rapide

  1. Installez SocialEraser depuis la section de téléchargement de notre page d'accueil.
  2. Cliquez sur l'icône SocialEraser dans la barre d'outils de Chrome pour ouvrir le panneau latéral.
  3. Ouvrez la plateforme que vous voulez nettoyer (X, TikTok, etc.) et connectez-vous à votre compte.
  4. Dans le panneau latéral, sélectionnez les éléments à nettoyer pour votre plateforme.
  5. Cliquez sur Démarrer le nettoyage. La barre de progression et les journaux s'affichent en direct.
  6. Utilisez Pause / Reprendre / Arrêter à tout moment.

FAQ

SocialEraser est-il sûr ? Mon compte peut-il être banni ?

SocialEraser effectue les mêmes actions que celles que vous feriez manuellement (cliquer sur « Supprimer », « Retirer le like », « Supprimer le signet », « Se désabonner »). Comme les actions sont de véritables événements du navigateur, elles paraissent identiques à des clics manuels. Cependant, les systèmes automatisés de X / Twitter peuvent quand même détecter une activité à fort volume. Nous recommandons :

  • Avancer à un rythme régulier — SocialEraser se met automatiquement en pause en cas de détection de limite de débit.
  • Éviter de laisser l'extension tourner pendant de nombreuses heures d'affilée.
  • Utiliser les filtres de date / mot-clé pour nettoyer par lots plus petits et ciblés.

Vous assumez l'entière responsabilité du respect des Conditions d'utilisation de X. Voir nos Conditions d'utilisation pour plus de détails.

Pourquoi certains de mes tweets / likes ne sont-ils pas supprimés ?

Raisons courantes :

  • Tweets épinglés — Les tweets épinglés ne peuvent pas être supprimés depuis l'interface utilisateur. Désépinglez-les d'abord.
  • Retweets de comptes suspendus — ils ne peuvent pas être annulés via l'interface utilisateur.
  • Tweets d'un autre compte — assurez-vous d'être connecté au bon compte.
  • L'interface a changé — X.com met fréquemment à jour son interface. Cliquez sur l'icône d'actualisation dans le panneau latéral pour récupérer la dernière configuration de sélecteurs.

Puis-je supprimer des messages directs (DM) ?

Non. La suppression des DM de X / Twitter nécessite un véritable clic droit natif (menu contextuel), protégé par la vérification event.isTrusted du navigateur. Les événements injectés par JavaScript sont rejetés. La seule façon de contourner cela exigerait la permission chrome.debugger, que nous refusons de demander parce qu'elle déclenche un avertissement fort de confiance utilisateur. Pour supprimer des DM, utilisez l'interface native de X manuellement.

Où sont stockées mes données ?

100% sur votre appareil, dans le stockage local de Chrome. L'extension n'a pas de serveur. Désinstaller l'extension supprime définitivement toutes les données stockées. Voir notre Politique de confidentialité pour tous les détails.

Quelles langues sont prises en charge ?

Actuellement : English, 简体中文, 日本語, 한국어, Português, Español, Deutsch, Français. Basculez via l'icône du drapeau dans le panneau latéral.

SocialEraser fonctionne-t-il sur mobile ?

Une version Android (Capacitor) est en cours de développement. iOS est prévu pour plus tard. Inscrivez-vous à la liste d'attente à support@socialeraser.app.

Et TikTok / YouTube / Facebook ?

Bientôt disponible. Le projet est en développement actif. Suivez notre GitHub pour les actualités.

Comment désinstaller ?

Faites un clic droit sur l'icône SocialEraser dans la barre d'outils de Chrome → « Supprimer de Chrome ». Cela supprime toutes les données stockées localement.

Signaler un bug

Vous avez trouvé un bug ou avez une demande de fonctionnalité ? Ouvrez un ticket sur GitHub ou envoyez un e-mail à support@socialeraser.app.
