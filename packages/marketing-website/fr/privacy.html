Politique de confidentialité

Dernière mise à jour : 20 juin 2026  ·  Effet immédiat

TL;DR : SocialEraser fonctionne 100% en local dans votre navigateur ou votre application mobile. Nous ne collectons, transmettons, stockons ni ne vendons aucune de vos données. Il n'y a pas de serveur. Désinstaller l'extension supprime définitivement toutes les préférences stockées.

1. Vue d'ensemble

SocialEraser (« nous », « notre ») fournit une extension de navigateur et des applications mobiles qui vous aident à supprimer en masse publications, likes, abonnements et éléments sauvegardés sur les plateformes de réseaux sociaux (actuellement X / Twitter ; prise en charge future de TikTok, YouTube, Facebook). La présente Politique de confidentialité explique quelles données l'extension manipule et comment.

2. Données que nous ne collectons PAS

  • ❌ Nous ne collectons pas votre nom d'utilisateur, mot de passe ou identifiants de connexion.
  • ❌ Nous ne collectons pas vos publications, likes, abonnements ou éléments sauvegardés.
  • ❌ Nous ne collectons pas votre adresse IP, empreinte de navigateur ou identifiants d'appareil.
  • ❌ Nous n'utilisons pas d'analyse, pixels de suivi ou scripts tiers.
  • ❌ Nous ne vendons ni ne partageons aucune donnée avec des tiers.
  • ❌ Nous n'opérons aucun serveur backend. Il n'y a rien à pirater.

3. Données stockées localement (sur VOTRE appareil)

L'extension utilise chrome.storage.local pour stocker uniquement les préférences non sensibles suivantes, qui restent toutes sur votre appareil :

  • Préférence de langue — laquelle des 8 langues prises en charge vous avez sélectionnée.
  • Compteur d'utilisation quotidien — combien d'actions de nettoyage vous avez effectuées aujourd'hui (pour l'analyse d'utilisation et la détection de limite de débit).
  • Reçu de la tirelire (le cas échéant) — cache local de votre dernière confirmation de don, si vous choisissez de soutenir le projet.
  • Cache de la configuration à distance — une copie de la dernière configuration des sélecteurs de plateforme (par ex. les sélecteurs CSS pour les boutons « Supprimer »), rafraîchie toutes les 24 heures.

Aucune de ces données ne quitte votre appareil. Vous pouvez l'inspecter via chrome://extensions → SocialEraser → « service worker » → Storage de Chrome. Désinstaller l'extension supprime définitivement toutes les données stockées.

4. Permissions expliquées

Les permissions ci-dessous s'appliquent uniquement à l'extension du navigateur Chrome. Le site web SocialEraser (cette page) et les applications Android / iOS prévues ne demandent aucune permission au niveau du navigateur.

L'extension demande les permissions Chrome suivantes, chacune justifiée ci-dessous :

PermissionPourquoi nous en avons besoin
storageStocker vos préférences localement (voir section 3).
activeTabLire la page x.com actuelle quand vous cliquez sur « Démarrer le nettoyage ».
sidePanelAfficher l'interface de nettoyage dans le panneau latéral de Chrome.
tabsNaviguer entre les onglets /likes, /bookmarks, /following, /with_replies pour nettoyer chaque section.
scriptingInjecter le moteur de nettoyage dans les pages x.com.
<all_urls>Accorde au service worker en arrière-plan l'accès hôte nécessaire pour appeler chrome.scripting.executeScript() sur l'onglet actif de l'utilisateur. Le content script lui-même est limité à x.com / twitter.com par le champ content_scripts.matches du manifest — l'extension ne s'exécute jamais sur un autre domaine.

5. Comment fonctionne l'extension

  1. Vous cliquez sur l'icône SocialEraser dans Chrome, ce qui ouvre le panneau latéral.
  2. Vous sélectionnez les éléments à nettoyer (publications / likes / abonnements / éléments sauvegardés).
  3. Vous cliquez sur Démarrer le nettoyage.
  4. Le content script de l'extension, s'exécutant dans x.com, simule des clics utilisateur sur les boutons « Supprimer » / « Retirer le like » / « Supprimer le signet » / « Se désabonner » — les mêmes boutons que vous cliqueriez manuellement.
  5. Tous les clics se produisent dans votre navigateur, sur votre appareil. Aucune donnée n'est envoyée ailleurs.

6. Services tiers

L'extension ne s'intègre à aucun service tiers d'analyse, de publicité, de suivi d'erreurs ou de télémétrie. Nous n'utilisons ni Google Analytics, ni Sentry, ni Mixpanel, ni Plausible, ni aucun outil similaire.

7. Configuration à distance

Pour gérer les changements d'interface de X.com sans vous forcer à mettre à jour l'extension, l'extension récupère périodiquement un petit fichier JSON (configuration des sélecteurs) depuis socialeraser.app. Cette récupération est anonyme (aucun identifiant utilisateur n'est envoyé) et ne contient aucune donnée personnelle. Vous pouvez désactiver le rafraîchissement de la configuration à distance en désactivant votre connexion réseau ; l'extension utilisera alors les sélecteurs de secours intégrés.

8. Enfants

L'extension n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons sciemment aucune donnée auprès d'enfants. Si vous pensez qu'un enfant a utilisé l'extension, veuillez nous contacter afin que nous puissions répondre à votre préoccupation.

9. Utilisateurs internationaux (GDPR / CCPA / LGPD)

Parce que nous ne collectons aucune donnée, les droits des personnes concernées par le GDPR / CCPA / LGPD (accès, suppression, portabilité) sont trivialement satisfaits : il n'y a rien à accéder, à supprimer ou à exporter. Si vous avez des préoccupations, contactez-nous à l'adresse ci-dessous.

10. Modifications de cette politique

Nous pouvons mettre à jour cette Politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date « Dernière mise à jour ». Si les modifications sont importantes, nous les noterons dans les notes de version de l'extension et (pour la prochaine mise à jour) à l'intérieur de l'interface du panneau latéral.

11. Contact

Questions, préoccupations ou demandes ? Envoyez un e-mail à support@socialeraser.app. Nous répondons sous 7 jours ouvrés.

12. Open source

Le code source de l'extension est disponible sur github.com/socialeraser/SocialEraser. Vous êtes invités à auditer le code, à le compiler vous-même et à vérifier nos affirmations en matière de confidentialité.
