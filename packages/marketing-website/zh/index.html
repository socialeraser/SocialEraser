The Basics

What exactly does SocialEraser do?

It bulk-deletes content on the social media platform of your choice — posts, likes, follows, saved items, and more. You pick what to clean, set filters, and it processes items one by one in your browser, with a pause / resume state machine.

Is SocialEraser really free?

Yes — SocialEraser is tip-supported. The free tier is the version, with no limits, no accounts, no trial. A tip jar is welcome but never required. No paywall, no quota, no upgrade prompts.

Do I need to create an account?

No. Zero accounts, zero signups, zero emails. Install, use, done.

Which browsers do you support?

Chrome and Edge today. Firefox and Safari are on the roadmap for 2026.

Do I need to keep my browser open during cleanup?

The browser tab where the cleanup runs must stay open, but you can minimize it. If you close Chrome, the run pauses and resumes exactly where it left off when you reopen. The Side Panel can be detached from the tab so it stays visible.

Privacy & Security

Do you ask for my password?

Never. SocialEraser never asks for your social media password. It runs inside the browser session you're already logged into. We can't see your account because we don't have credentials to see it with.

Do you collect my data?

No. We have no servers that receive your data. The extension runs entirely client-side. The only thing stored in chrome.storage is your preferences (language, daily counter) — and you can wipe that with one click.

Is it safe to install?

The extension is reviewed by Google before it lands in the Chrome Web Store. The source code is open source on GitHub (MIT licensed), so anyone can audit it. We use a strict Content Security Policy that blocks all remote scripts.

Can my account get banned for using this?

SocialEraser simulates normal clicks at a human-like rate. The platforms have rate limits, and we respect them. That said, we are not affiliated with any platform — using bulk tools is at your own discretion. The same applies to any third-party tool.

Can I verify the extension doesn't talk to a server?

Yes. Open Chrome DevTools → Network tab while the extension is running, and you'll see zero outbound requests to any third-party domain. The only network activity is between your browser and the social platform you're cleaning — the same requests you'd make manually. The full source is on GitHub for anyone to audit.

How It Works

How does bulk delete actually work?

The extension opens your platform's native UI (e.g. x.com), finds the action menu for each item, clicks delete, and confirms. It's the same as you doing it by hand — just faster and tireless.

What filters are available?

Date range (delete posts before / after X), keyword (only delete posts containing "Y"), media type (only images, only text, only videos), and account (only delete from this handle). Filter combinations work as AND, not OR.

Can I pause and resume a run?

Yes. Pause / resume works any time, even after closing the browser. State is stored in chrome.storage.local, so it survives a restart. Come back the next day and continue exactly where you left off.

How fast is it?

1 件平均不到 1 秒。1000 条的处理时间约为 15 分钟。准确估算可使用上方的 时间计算器。

Can I export before deleting?

Yes. The X-tier extension has an "Export archive" option that saves an archive.json of every item before deletion. We're adding the same option to other platforms as they ship.

What happens if my internet drops mid-cleanup?

The extension detects the disconnect, pauses the run, and shows a "Reconnecting…" indicator. Once the network is back, it resumes from the last completed item. Nothing is sent or stored anywhere — the resume state lives only in your browser's local storage.

Can I undo a cleanup?

No — once a post is deleted on the platform, it's gone. We recommend using the "Export archive" option (where available) to save a JSON backup before running a clean, so you have a record of what was removed.

vs. The Alternatives

SocialEraser vs TweetDelete?

TweetDelete uses the X API and requires you to authenticate with OAuth. It works server-side, so your data leaves your machine. SocialEraser runs in-browser, never asks for credentials, and works on 5 platforms — not just X.

SocialEraser vs Semiphemeral?

Semiphemeral is auto-deletion based on age — it deletes tweets older than X months on a schedule. SocialEraser is on-demand bulk deletion — you choose what to clean, when, and how much. Different tools for different jobs.

SocialEraser vs Redact?

Redact is a paid dev tool that deletes tweets via the API in bulk. It requires API keys and a developer account. SocialEraser is free, requires no API access, and works on platforms beyond just X.

SocialEraser vs doing it manually?

Manually deleting 1,000 items takes roughly 3 hours of continuous clicking. SocialEraser does the same job in about 15 minutes — and it never gets tired, never misses a confirmation dialog, and never accidentally clicks the wrong button at 2am. It's the same actions, just automated.

Per-Platform

YouTube / Instagram / Facebook 什么时候上线？

X 和 TikTok 已上线。YouTube、Instagram、Facebook 各有不同的 UI 模式、速率限制和反爬虫措施 — 我们一次只发一个平台，确保稳定可靠后再开放。加入 YouTube 候补名单，上线时第一时间收到通知。

Will mobile (iOS / Android) apps be free too?

Yes — the Android and iOS apps will be tip-supported as well, same model as desktop. No limits, no accounts, no paywalls. The free tier is the version.

Can I run SocialEraser on multiple accounts?

On Chrome, you can use separate browser profiles for each account. Multi-account within a single profile is on the roadmap.

Do I need to log into the platform first?

Yes — SocialEraser works inside the browser session you're already logged into. It does not log in for you, store cookies, or touch your auth tokens. If your session expires, simply log in again and resume the run.
