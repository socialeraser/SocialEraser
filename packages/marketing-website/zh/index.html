基础

SocialEraser 具体能做什么？

它能批量删除你所选社交媒体平台上的内容 — 帖子、点赞、关注、收藏等。你选择要清理的内容、设置过滤条件，然后在你的浏览器中逐条处理，自带暂停 / 恢复状态机。

SocialEraser 真的免费吗？

是的——SocialEraser 是打赏支持型工具。免费版就是完整版，没有次数限制、没有账号要求、没有试用期。欢迎打赏，但从不强求。没有付费墙、没有配额、没有升级提示。

需要注册账号吗？

不需要。零账号、零注册、零邮件。装上就能用，就这么简单。

支持哪些浏览器？

目前支持 Chrome 和 Edge。Firefox 和 Safari 已列入 2026 年路线图。

清理过程中需要保持浏览器打开吗？

运行清理的浏览器标签页必须保持打开，但可以最小化。如果关闭 Chrome，任务会暂停，重新打开后会从上次中断的位置继续。侧边栏可以从标签页分离，关闭标签页时仍然可见。

隐私与安全

会要求输入密码吗？

绝不会。SocialEraser 从不索取你的社交媒体密码。它运行在你已经登录的浏览器会话中。我们看不到你的账户，因为我们没有用于查看的凭据。

你们会收集我的数据吗？

不会。我们没有接收你数据的服务器。扩展完全在客户端运行。唯一存储在 chrome.storage 中的是你的偏好设置（语言、每日计数）— 你可以一键清除。

安装安全吗？

扩展在上架 Chrome 应用商店前会经过 Google 审核。源代码已在 GitHub 开源（MIT 协议），任何人都可以审计。我们使用严格的内容安全策略，阻止所有远程脚本。

使用这个工具会导致我的账户被封吗？

SocialEraser 以接近人类的速度模拟正常点击。各平台都有速率限制，我们都会遵守。话虽如此，我们与任何平台都没有关联 — 使用批量工具的风险请自行判断。这一点对任何第三方工具都一样。

如何验证扩展不会与外部服务器通信？

可以验证。扩展运行时打开 Chrome DevTools → Network 标签页，你会看到对任何第三方域名的出站请求为零。唯一的网络活动发生在你的浏览器和正在清理的社交平台之间 — 跟你手动操作时的请求完全一致。完整源码已在 GitHub 公开，欢迎任何人为之审计。

工作原理

批量删除具体是怎么工作的？

扩展打开平台的原生界面（如 x.com），为每条内容找到操作菜单，点击删除，然后确认。这和你手动操作完全一样 — 只是更快，而且不知疲倦。

有哪些过滤选项？

日期范围（删除 X 之前 / 之后的帖子）、关键词（只删除包含 "Y" 的帖子）、媒体类型（只要图片、只要文字、只要视频），以及账号（只删除该账号的）。多个过滤条件之间是 AND 关系，不是 OR。

可以暂停然后继续吗？

可以。暂停 / 继续随时可用，即使关闭浏览器也行。状态保存在 chrome.storage.local，重启后也能保留。第二天回来，从上次离开的位置继续。

批量删除到底有多快？

1 件平均不到 1 秒。1000 条的处理时间约为 15 分钟。准确估算可使用上方的 时间计算器。

删除前能先导出吗？

可以。X 版扩展提供「导出归档」选项，会在删除前保存一份 archive.json。我们正在为其他平台陆续添加这个功能。

清理中途断网会怎样？

扩展会检测到断连，暂停任务，并显示「正在重新连接…」提示。网络恢复后，从上次完成的条目继续。不会有任何数据被发送或存储到别处 — 恢复状态只保存在你浏览器的本地存储中。

清理后能撤销吗？

不能 — 一旦帖子在平台上被删除，就无法恢复。我们建议在运行清理前使用「导出归档」选项（如果可用）保存一份 JSON 备份，留一份被删除内容的记录。

与替代方案对比

SocialEraser vs TweetDelete？

TweetDelete 使用 X API，需要你用 OAuth 授权。它在服务端运行，所以你的数据会离开本机。SocialEraser 在浏览器内运行，从不索取凭据，支持 5 个平台 — 不仅仅是 X。

SocialEraser vs Semiphemeral？

Semiphemeral 是基于时间的自动删除 — 它按计划删除超过 X 个月的推文。SocialEraser 是按需批量删除 — 你来决定清理什么、何时清理、清理多少。不同工具适用于不同场景。

SocialEraser vs Redact？

Redact 是一款付费的开发者工具，通过 API 批量删除推文。它需要 API 密钥和开发者账号。SocialEraser 免费，无需 API 访问，而且支持 X 之外的更多平台。

SocialEraser vs 手动操作？

手动删除 1000 条内容大约需要连续点击 3 小时。SocialEraser 能在 15 分钟左右完成同样的工作 — 而且它不会累、不会漏掉确认弹窗、不会在凌晨 2 点不小心点错按钮。动作是一样的，只是自动化的。

各平台问题

YouTube / Instagram / Facebook 什么时候上线？

X 和 TikTok 已上线。YouTube、Instagram、Facebook 各有不同的 UI 模式、速率限制和反爬虫措施 — 我们一次只发一个平台，确保稳定可靠后再开放。加入 YouTube 候补名单，上线时第一时间收到通知。

移动端 (iOS / Android) 应用也免费吗？

是的——Android 和 iOS 应用同样是打赏支持型，与桌面端同一个模式。没有次数限制、没有账号要求、没有付费墙。免费版就是完整版。

能在多个账号上使用 SocialEraser 吗？

在 Chrome 上，你可以为每个账号使用单独的浏览器配置文件。单个配置文件内的多账号支持已列入路线图。

需要先登录平台吗？

需要 — SocialEraser 在你已经登录的浏览器会话中运行。它不会替你登录、不存储 cookie、也不触碰你的认证令牌。如果会话过期，简单地重新登录再继续任务即可。
