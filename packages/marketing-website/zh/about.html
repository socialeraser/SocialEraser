关于 SocialEraser

由一个厌倦了点击一千次「删除」的人打造。

故事

几年前，我想清理一下自己的旧推文。没什么大事——只是一些过时的圈内梗、半成型的想法， 和那些到 2017 年左右就不再好笑的截图。我原以为这事儿几分钟就能搞定，也许最多一个小时。

结果花了三天。

X 让我一条一条地删除推文。每一条都要：滚动、点击菜单、点击「删除」、点击「确认」、等待页面刷新、再滚动、再来一遍。 我有超过 4000 条推文。按每条约 10 秒算，那就是十一个小时的点击。我第一天下午就放弃了。

我试了试现有的工具。它们大多数都要我的密码、我的 API key，或者我的信用卡。 少数不要的，也仍然把所有操作都过一遍它们的服务器——也就是说，每一条点赞、每一个关注、每一条私信， 都得先跑到别人的数据库里转一圈，才能被删掉。我对此不太放心。 我不应该非得相信一个陌生人的数据库，才能清理我自己的账号。

所以我做了 SocialEraser。

我们的信念

  • 你的数据是你的。如果一个工具必须拿到你的密码才能清理你的账号，那这个设计从一开始就跑偏了。
  • 没有服务器比任何服务器都安全。「无服务器」架构是做出「永远可以坚守的隐私承诺」的唯一办法。
  • 免费就应该是免费。没有试用期，没有配额，没有每日上限。打赏欢迎，但从不强求。
  • 开源即是问责。任何人都可以审计代码、提 issue、fork 项目。隐私工具里不该有秘密。
  • 慢一点没关系。批量删除不是赛跑。我们把节奏调成「一个谨慎的人会做的那样」——这也是账号不会出问题的关键。

我们不是的东西

我们与 X、TikTok、YouTube、Instagram 或 Facebook 没有任何关联——以后也不会有。 我们不卖你的数据（我们根本没有）。我们不打广告。我们不接受任何我们使用了它们 API 的平台的投资。 如果这些情况中有任何一项发生变化，我们会提前、公开、大声地宣布，而不是事后再说。

我们是一个非常小的团队——目前只有一位开发者，有时候两位——在做我们希望几年前就存在的东西。 扩展是免费的，代码在 GitHub 上公开， 我们唯一想要的回报是：如果它帮到了你，请告诉一个朋友。
