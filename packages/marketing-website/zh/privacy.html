Privacy Policy（隐私政策）

最后更新：2026 年 6 月 20 日  ·  即时生效

TL;DR：SocialEraser 100% 在你的浏览器或移动 App 中本地运行。 我们不收集、不传输、不存储、也不出售你的任何数据。我们没有服务器。 卸载扩展会永久删除所有存储的偏好设置。

1. 概述

SocialEraser（以下简称「我们」）提供一款浏览器扩展和移动应用， 帮助你批量删除社交媒体平台（目前支持 X / Twitter；未来支持 TikTok、YouTube、Facebook）上的帖子、点赞、关注和收藏。 本 Privacy Policy 说明了扩展所处理的数据以及处理方式。

2. 我们不收集的数据

  • ❌ 我们不收集你的用户名、密码或登录凭证。
  • ❌ 我们不收集你的帖子、点赞、关注或收藏内容。
  • ❌ 我们不收集你的 IP 地址、浏览器指纹或设备标识符。
  • ❌ 我们不使用分析、追踪像素或第三方脚本。
  • ❌ 我们不向第三方出售或共享任何数据。
  • ❌ 我们不运营任何后端服务器——没有什么可以被攻破。

3. 本地存储的数据（在你的设备上）

扩展使用 chrome.storage.local 仅存储以下非敏感偏好，所有数据都保留在你的设备上：

  • 语言偏好 — 你选择的 8 种支持语言之一。
  • 每日使用计数 — 你今天已执行的清理操作数量（用于使用统计与速率限制检测）。
  • 打赏凭证（如有） — 你上一次打赏确认信息的本地缓存，如果你选择支持本项目的话。
  • 远程配置缓存 — 最新平台选择器配置的副本（例如「删除」按钮的 CSS 选择器），每 24 小时刷新一次。

这些数据都不会离开你的设备。你可以通过 Chrome 的 chrome://extensions → SocialEraser → 「service worker」→ Storage 查看它们。 卸载扩展会永久删除所有存储的数据。

4. 权限说明

以下权限仅适用于 Chrome 浏览器扩展。SocialEraser 网站（当前页面） 以及计划推出的 Android / iOS 应用都不会请求任何浏览器级别的权限。

扩展请求以下 Chrome 权限，每项权限都有对应说明：

权限申请原因
storage在本地存储你的偏好设置（见第 3 节）。
activeTab在你点击「Start Cleanup」时，读取当前 X.com 页面。
sidePanel在 Chrome 侧边面板中显示清理 UI。
tabs在 /likes、/bookmarks、/following、/with_replies 标签之间切换以清理每个区域。
scripting将清理引擎注入 x.com 页面。
<all_urls>授予后台 Service Worker 调用 chrome.scripting.executeScript() 时所需的主机访问权限。内容脚本本身受 manifest 的 content_scripts.matches 字段限制，只能在 x.com / twitter.com 域运行——扩展不会在任何其他域运行。

5. 扩展的工作方式

  1. 你点击 Chrome 中的 SocialEraser 图标，打开侧边面板。
  2. 你选择要清理的内容（帖子 / 点赞 / 关注 / 收藏）。
  3. 你点击 Start Cleanup。
  4. 扩展的内容脚本在 x.com 内部运行，模拟用户点击「Delete」/「Unlike」/「Remove Bookmark」/「Unfollow」按钮——也就是你手动点击的同一个按钮。
  5. 所有点击都在你的浏览器中、你的设备上发生。没有任何数据被发送到任何地方。

6. 第三方服务

本扩展不集成任何第三方分析、广告、错误追踪或遥测服务。 我们不使用 Google Analytics、Sentry、Mixpanel、Plausible 或任何类似工具。

7. 远程配置

为了在不强制你更新扩展的情况下处理 X.com 的 UI 变化，扩展会定期从 socialeraser.app 拉取一个小的 JSON 文件（选择器配置）。 该请求是匿名的（不发送任何用户标识符），不包含任何个人数据。 你可以通过关闭网络连接来禁用远程配置刷新；扩展随后将使用打包的备用选择器。

8. 儿童

本扩展不面向 13 岁以下儿童。我们不故意收集儿童的任何数据。 如果你认为有儿童使用了本扩展，请联系我们以便我们处理你的疑虑。

9. 国际用户（GDPR / CCPA / LGPD）

由于我们不收集数据，GDPR / CCPA / LGPD 规定的数据主体权利（访问、删除、可携性）可以自然得到满足： 没有什么可以访问、删除或导出的。如果你有任何顾虑，请通过下方地址联系我们。

10. 本政策的变更

我们可能会更新本 Privacy Policy。变更将发布在此页面，并更新「最后更新」日期。 如果是重大变更，我们会在扩展的发布说明以及（下一次更新后）侧边面板 UI 中标注。

11. 联系方式

有问题、顾虑或请求？请发邮件到 support@socialeraser.app。 我们会在 7 个工作日内回复。

12. 开源

扩展的源代码可在 github.com/socialeraser/SocialEraser 获取。 欢迎审计代码、自行构建并验证我们的隐私声明。
