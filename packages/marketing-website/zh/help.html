帮助与常见问题

最后更新：2026 年 6 月 20 日

快速上手

  1. 从 首页的下载区安装 SocialEraser。
  2. 在 Chrome 工具栏点击 SocialEraser 图标，打开侧边面板。
  3. 打开你想清理的平台（X、TikTok 等）并登录你的账号。
  4. 在侧边面板中，选择要为你所在平台清理的内容类型。
  5. 点击 开始清理。进度条和日志会实时显示。
  6. 随时可以使用 暂停 / 继续 / 停止。

常见问题

SocialEraser 安全吗？我的账号会被封吗？

SocialEraser 执行的操作和你手动操作完全一样（点击「删除」、「取消点赞」、「移除书签」、「取消关注」）。 由于这些操作都是真实的浏览器事件，从系统上看与手动点击毫无区别。 不过，X / Twitter 的自动化系统仍可能检测到高频活动。我们建议：

  • 保持稳定节奏运行 —— SocialEraser 在检测到速率限制时会自动暂停。
  • 避免长时间连续运行扩展。
  • 使用日期 / 关键词筛选，按更小、更精准的批次进行清理。

你需自行负责确保遵守 X 的服务条款（Terms of Service）。详情请参阅我们的 服务条款。

为什么部分推文 / 点赞没有被删除？

常见原因：

  • 置顶推文 —— 置顶推文无法在 UI 中删除，请先取消置顶。
  • 来自已封禁账号的转发 —— 这些无法通过 UI 撤销。
  • 来自其他账号的推文 —— 请确认你登录的是正确的账号。
  • UI 变更 —— X.com 经常更新 UI。点击侧边面板中的刷新图标即可获取最新的选择器配置。

我能删除私信（DM）吗？

不能。X / Twitter 的 DM 删除需要真实的原生右键（上下文菜单）操作， 而这受到浏览器 event.isTrusted 检查的保护。JavaScript 注入的事件会被拒绝。 唯一能绕过这个限制的办法需要 chrome.debugger 权限，但我们不会申请这个权限， 因为它会触发强烈的用户信任警告。要删除 DM，请使用 X 自带的原生界面手动操作。

我的数据存储在哪里？

100% 存储在你本地的 Chrome 本地存储中。扩展没有服务器。卸载扩展会永久删除所有存储的数据。 完整细节请参阅我们的隐私政策。

支持哪些语言？

目前支持：English、简体中文、日本語、한국어、Português、Español、Deutsch、Français。 可通过侧边面板中的旗帜图标切换。

SocialEraser 能在移动端使用吗？

Android（Capacitor）版本正在开发中。iOS 已列入未来计划。 请在 support@socialeraser.app 提交邮箱加入候补名单。

TikTok / YouTube / Facebook 呢？

即将推出。项目正在积极开发中。关注我们的 GitHub 获取最新进展。

如何卸载？

右键 Chrome 工具栏中的 SocialEraser 图标 → 「从 Chrome 中移除」。 这将删除所有本地存储的数据。
