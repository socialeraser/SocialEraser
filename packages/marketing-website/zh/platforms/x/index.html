过滤条件

精准清理，而非盲目删除

批量的意义在于高效，但精准同样重要。组合使用过滤条件，只删你真正想删的内容。

日期范围

删除早于或晚于指定日期的推文。保留疫情期间的感悟，删除 2014 年的黑历史。最常用的过滤条件。

关键词匹配

仅删除包含特定词汇或短语的推文。非常适合清理某个话题 — 例如求职期、某场争论或某个饭圈话题。

媒体类型

只删除带图片的推文、只删除视频、只删除链接、或只删除纯文字。便于在删除前先备份媒体内容。

互动量阈值

保留点赞或回复超过 X 条的推文，清理那些无人问津的 0 互动内容。设为 0 则全部删除。

常见问题

X 相关问题

清理范围

我可以批量删除旧推文吗？

可以。设置日期过滤（例如2020 年 1 月 1 日之前），SocialEraser 会删除所有匹配的推文。每条约 0.9 秒 — 1,000 条推文大约需要 15 分钟。

我只能删除自己的回复吗？

可以。使用帖子类型过滤只针对回复（也可以只针对原创推文或转发），可与日期、关键词过滤组合使用。

我能保留点赞多的推文吗？

可以。使用互动量阈值过滤设置最低点赞数，高于阈值的推文会被保留，低于阈值的会被删除。

长推文（线程）怎么办？

SocialEraser 逐条删除推文。如果某条推文属于线程的一部分，其回复链会变成孤立内容但仍然可见。如需完全删除整个线程，请使用日期过滤按倒序删除推文。

安全

我的账号会被封禁吗？

SocialEraser 以人工速度模拟正常点击，并遵守速率限制。和所有第三方工具一样 — 我们与 X Corp. 没有任何关联，使用批量工具的风险由您自行承担。

我能撤销删除吗？

不能。推文一旦删除，几秒之内就会从 X 公开视图中消失。我们强烈建议在首次批量操作前使用导出归档功能保存备份。

如果 X 更改了界面怎么办？

SocialEraser 从远程 CDN 获取 DOM 选择器架构。当 X 发布 UI 变更时，我们会推送更新 — 无需重新安装扩展。更新是自动且静默的。

技术

它能在新的 x.com 界面上工作吗？

可以。SocialEraser 同时支持x.com和旧版twitter.com域名。选择器架构每周更新以匹配 X 最新的 UI 变化。

它能在移动端（手机浏览器中的 x.com）使用吗？

Android 上的 Chrome 浏览器支持大部分功能。iOS Safari 由于苹果的扩展机制限制较大 — 专门的 iOS 应用已列入 2027 路线图。

我能在多个 X 账号上运行吗？

可以 — 为每个账号使用单独的浏览器配置文件。单个配置文件内的多账号功能正在规划中。
