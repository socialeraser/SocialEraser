Quick Start

  1. Install SocialEraser from the Chrome Web Store.
  2. Click the SocialEraser icon in Chrome's toolbar to open the Side Panel.
  3. Open x.com and log in to your account.
  4. In the Side Panel, select which items to clean (Tweets / Likes / Bookmarks / Following).
  5. Click Start Cleanup. The progress bar and logs will appear live.
  6. Use Pause / Resume / Stop at any time.

FAQ

Is SocialEraser safe? Will my account be banned?

SocialEraser performs the same actions you would perform manually (clicking "Delete", "Unlike", "Remove Bookmark", "Unfollow"). Because the actions are real browser events, they appear identical to manual clicks. However, X / Twitter's automated systems may still detect high-volume activity. We recommend:

  • Stay within the 50-actions-per-day free limit (or upgrade to Premium when available).
  • Avoid running the extension for many hours continuously.
  • Use the date / keyword filters to clean in smaller, targeted batches.

You assume all responsibility for compliance with X's Terms of Service. See our Terms of Service for details.

Why are some of my tweets / likes not being deleted?

Common reasons:

  • Pinned tweets — Pinned tweets cannot be deleted from the UI. Unpin them first.
  • Retweets from accounts that have been suspended — these cannot be undone via the UI.
  • Tweets from a different account — make sure you're logged in to the correct account.
  • UI changed — X.com updates its UI frequently. Click the refresh icon in the Side Panel to fetch the latest selector config.

Can I delete Direct Messages (DMs)?

No. X / Twitter's DM deletion requires a real native right-click (context menu), which is protected by the browser's event.isTrusted check. JavaScript-injected events are rejected. The only way to bypass this would require the chrome.debugger permission, which we will not request because it triggers a strong user trust warning. To delete DMs, use X's native interface manually.

Where is my data stored?

100% on your device, in Chrome's local storage. The extension does not have a server. Uninstalling the extension permanently deletes all stored data. See our Privacy Policy for full details.

What languages are supported?

Currently: English, 简体中文, 繁體中文, 日本語, 한국어, Español, Deutsch, Français. Switch via the flag icon in the Side Panel.

Does SocialEraser work on mobile?

An Android (Capacitor) version is in development. iOS is planned for the future. Sign up for the waitlist at hello@socialeraser.app.

What about TikTok / YouTube / Facebook?

Coming soon. The project is in active development. Follow our GitHub for updates.

How do I uninstall?

Right-click the SocialEraser icon in Chrome's toolbar → "Remove from Chrome". This deletes all locally stored data.
